## 1. 背景与问题

### 1.1 当前问题

债券上攻排行需要显示每只债券的行业信息，当前实现存在性能问题：

```python
# 原逻辑：O(n*m) 双重循环反查
all_mappings = cache.get_all_mapping()  # 获取全部映射（数千条）
for bond_code in bond_codes:            # 遍历债券列表
    for stock_code, mapping in all_mappings.items():  # 遍历全部映射
        if mapping.get('bond_code') == bond_code:
            industry_name = mapping.get('industry_name')
```

**性能问题**:
- 每次请求都要遍历全部映射数据
- 时间复杂度 O(n*m)，n=债券数，m=映射总数
- 60只债券 × 3000条映射 = 18万次比较

### 1.2 优化目标

- 时间复杂度从 O(n*m) 降到 O(1)
- 查询延迟从 ~100ms 降到 ~5ms
- 支持更高并发

---

## 2. 架构设计

### 2.1 核心思想

创建独立的 `bond_industry_mapping` 缓存：
- 以**债券代码**为 key（替代股票代码）
- 直接查询行业信息（无需反查）
- 依赖 `stock_bond_mapping` 数据

### 2.2 数据流

```
MySQL (data_bond_ths, data_industry_code_component_ths)
    ↓
stock_bond_industry_mapping.py (生成映射)
    ↓
stock_bond_mapping_cache (Redis Hash: stock_code -> data)
    ↓
bond_industry_cache (提取并转换)
    ↓
bond_industry_mapping (Redis Hash: bond_code -> data)
    ↓
monitor.py _get_bond_industry_batch (O(1)查询)
```

### 2.3 缓存结构

```
bond_industry_mapping:2026-04-02 (Hash)
  ├── 127001 -> {"bond_code":"127001","bond_name":"XX转债","industry_name":"银行"}
  ├── 127002 -> {"bond_code":"127002","bond_name":"YY转债","industry_name":"房地产"}
  └── ...

bond_industry_mapping:latest_date -> "2026-04-02"
bond_industry_mapping:meta -> {"created_at":"...","total_count":500}
```

## 6. 使用方式

### 6.1 缓存预热（自动）

```python
# app.py 启动时自动预热
from gs2026.dashboard2.cache import init_all_caches, cache_manager
init_all_caches()
cache_manager.warmup_all(sync_names=['red_list'])
```

### 6.2 业务代码调用

```python
# monitor.py
from gs2026.dashboard2.cache.bond_industry import get_cache

def _get_bond_industry_batch(bond_codes: list) -> dict:
    cache = get_cache()
    if not cache.ensure_cache():
        return {code: '-' for code in bond_codes}
    
    # O(1) 批量查询
    return cache.get_industries_batch(bond_codes)
```

## 7. 降级策略

| 场景 | 处理 | 返回值 |
|------|------|--------|
| 缓存未初始化 | 调用 ensure_cache() 自动创建 | 正常查询 |
| 缓存创建失败 | 降级返回默认值 | {code: '-'} |
| 债券不存在 | 返回 '-' | '-' |
| Redis 故障 | 异常捕获，返回默认值 | {code: '-'} |
