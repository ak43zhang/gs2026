# 公告分析V2 — 整体设计方案 V1.0

## 文档状态

首版本，等待用户审核后再实施。

## 一、设计目标

AI分析后的公告价值密度极高，但当前展示方式只是"列表+标签"，大量分析结果被埋没。

**核心目标：**
1. **快速定位顶级信号** — 几秒内找到"高影响力度+利好"和"高影响力度+利空"
2. **快速定位隔夜超短机会** — 直接看`市场预期`/`开盘预判`/`影响力/距今`，一眼可决策
3. **最大化利用AI分析字段** — 不再只显示标题和风险标签
4. **支撑顶级公告分析报告** — 精选公告的深度展示版面

## 四、核心方案设计

### 4.1 顶部：顶级信号快览区（新增）

**设计理念：** 把4个组合做成大尺寸卡片网格，一眼扫完关键信号。

**布局：**
```
┌─────────────────────────────────────────────────────────┐
│ 🚀极高利好信号                               🔴极高利空信号 │
│                                                         │
│ [股票名] [公告标题16字]                                  │
│ 影响: 重大 ⭐⭐⭐  预判: 大幅高开>5%  持续: 持续发酵     │
│ 公告类型: 业绩暴增超预期                                   │
│ 隔夜策略: 可以...                                        │
│ ─────────────────────────────────────────────────────── │
│ [股票名] [公告标题16字] [影响:重大 ⭐⭐⭐ 预判:大幅低开>5%] │
└─────────────────────────────────────────────────────────┘
```

**组合优先级排序：**
1. 🚀 **高+利好**（非常强利好，大量级隔夜机会）← 最大注意
2. 🔴 **高+利空**（非常强利空，大量级避雷）← 同等重要
3. 📈 **中+利好+超预期**（中力度但超预期，有溢价空间）
4. 📉 **中+利空+低于预期**（中力度但低于预期，有下杀风险）
5. 🔍 **高+中性**（重大事件中性看待，可能是事件驱动型）

**每个卡片展示字段优先级（由左至右重要性递减）：**
```
[股票名+代码] [公告标题18字]          ← 最需要一眼辨识
影响:重大 ⭐⭐⭐   预告:大幅高开>5%     ← 超短最核心判断
公告类型:业绩暴增超预期                 ← 做了什么
持续:持续发酵                          ← 能走多远
隔夜策略: 可隔夜追入...               ← 怎么操作（高信号专属）
```

## 六、详细字段说明（超短视角）

### 6.1 影响力度（一言判断）

| 值 | 含义 | 超短含义 |
|----|------|----------|
| **高** ⭐⭐⭐ | 足以单独驱动涨停/跌停 | 重大事件，隔夜跳空预期大 |
| **中** ⭐⭐ | 可能2~5%波动 | 有影响但不独立 |
| **低** ⭐ | 轻微或无实质影响 | 例行公告，忽略 |

### 6.2 市场预期（相对位置判断）

| 值 | 含义 | 超短含义 |
|----|------|----------|
| **超预期** | 好于/差于市场已有预期 | 会引发大量溢价/折价 |
| **符合预期** | 已被price-in | 反应温和 |
| **低于预期** | 不及市场期望 | 可能高开低走 |

### 6.3 开盘预判（7档跳空预判）

| 档位 | 利好含义 | 利空含义 |
|------|---------|---------|
| 大幅高开(>5%) | 超强信号 | — |
| 高开(2-5%) | 强信号 | — |
| 小幅高开(0-2%) | 正常 | — |
| 平开 | 影响有限 | — |
| 小幅低开(0-2%) | — | 偏弱 |
| 低开(2-5%) | — | 偏强 |
| 大幅低开(>5%) | — | 极强信号 |

### 6.4 持续性（阶段判断）

| 值 | 含义 | 操作含义 |
|----|------|----------|
| **一日游** | 次日影响最大，之后回归 | 隔夜博弈 |
| **2-3日** | 短期持续发酵 | 可延续持有 |
| **一周以上** | 中期影响 | 中期布局 |
| **持续发酵** | 基本面级别变化 | 长期重仓 |

### 6.5 隔夜策略（操作建议）

AI在Prompt中被要求从"是否值得隔夜介入/预期收益/风险点/建议仓位"四个维度给出建议。
