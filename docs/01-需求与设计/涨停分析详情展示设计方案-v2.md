## 二、详情面板布局

### 2.1 整体结构

```
┌─────────────────────────────────────────────────────────────────┐
│  列表页（左侧）              │  详情面板（右侧，宽度600px）       │
│                              │                                   │
│  ┌──────────────────┐       │  ┌─────────────────────────────┐  │
│  │ 涨停卡片1        │       │  │ 🔥 股票名称 (代码)      [×] │  │
│  │ 涨停卡片2        │       │  ├─────────────────────────────┤  │
│  │ ...              │       │  │                             │  │
│  │                  │       │  │  📊 核心指标                │  │
│  │                  │       │  │  🏭 板块与概念              │  │
│  │                  │       │  │  🐉 股性与龙虎榜            │  │
│  │                  │       │  │  📰 消息驱动                │  │
│  │                  │       │  │  🔮 预期与延续              │  │
│  │                  │       │  │  🔍 深度分析                │  │
│  │                  │       │  │                             │  │
│  └──────────────────┘       │  └─────────────────────────────┘  │
│                              │                                   │
└──────────────────────────────┴───────────────────────────────────┘
```

### 2.2 详情区块设计

#### 📊 核心指标区
- 涨停时间 + 时段
- 股性评分
- 有预期（是/否 + 消息数量）
- 延续性强（是/否）

#### 🏭 板块与概念区
- 所属板块列表
- 相关概念列表
- 龙头股列表

#### 🐉 股性与龙虎榜区
- 股性分析文本
- 龙虎榜分析文本

#### 📰 消息驱动区
- 板块消息（列表）
- 概念消息（列表）
- 龙头股消息（列表）
- 影响消息（列表）

#### 🔮 预期与延续区
- 预期涨停消息（列表，带延续性标记）
- 延续性分析

#### 🔍 深度分析区
- 技术面分析
- 基本面分析
- 资金面分析
- 情绪面分析

## 三、交互设计

1. **点击卡片**：右侧滑出详情面板（300ms动画）
2. **关闭面板**：点击×、点击外部、ESC键
3. **面板内滚动**：独立滚动条
4. **区块折叠**：每个区块可折叠/展开
