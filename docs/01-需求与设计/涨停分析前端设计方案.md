# 涨停分析前端页面设计方案

## 一、设计目标

1. **信息层次清晰**：列表页展示关键信息，详情页展示完整分析
2. **数据完整呈现**：所有分析字段都有对应的展示位置
3. **用户体验优化**：类似新闻分析的点击展开详情模式
4. **统计信息丰富**：多维度数据可视化

## 四、交互设计

### 4.1 列表页交互
1. **筛选**: 日期、股票、时段、预期、延续性
2. **排序**: 默认按涨停时间升序
3. **点击**: 点击卡片展开详情模态框
4. **分页**: 每页20条，滚动加载或分页按钮

### 4.2 详情页交互
1. **展开**: 点击列表项弹出模态框
2. **关闭**: 点击×或背景关闭
3. **导航**: 支持键盘ESC关闭
