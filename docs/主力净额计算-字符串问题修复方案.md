# 主力净额计算 - "-"字符串问题修复方案

## 问题分析

### 错误信息
```
计算主力净额失败: unsupported operand type(s) for -: 'str' and 'float'
```

### 问题根因

1. **数据来源**: `df_prev_main` 从 Redis 加载，包含历史计算结果
2. **字段值**: `cumulative_main_net` 和 `main_net_amount` 字段可能包含 "-" 字符串
3. **报错位置**: 数值计算时遇到"-"字符串导致类型错误

### 具体场景

| 场景 | 字段值 |
|------|--------|
| 首次计算 | 0（数值） |
| 历史"无主力" | "-"（字符串） |
| 历史有值 | 1234567.89（数值） |

## 修复方案

### 最终方案：双字段数据清洗

**规则**: 遇到"-"或其他非数值，统一设置为0

```python
# 修复代码
if 'cumulative_main_net' in df_prev_main.columns:
    # 【修复】将cumulative_main_net转为数值，"-"等非数值转为0
    df_prev_main['cumulative_main_net'] = pd.to_numeric(
        df_prev_main['cumulative_main_net'], 
        errors='coerce'
    ).fillna(0)

if 'main_net_amount' in df_prev_main.columns:
    # 【修复】将main_net_amount转为数值，"-"等非数值转为0
    df_prev_main['main_net_amount'] = pd.to_numeric(
        df_prev_main['main_net_amount'], 
        errors='coerce'
    ).fillna(0)
```
