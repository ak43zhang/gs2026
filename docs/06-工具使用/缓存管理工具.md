# 缓存管理工具使用说明

## cache_manager.py

管理内存缓存和宽表缓存

### 使用示例

```bash
# 预热宽表缓存
python tools/cache_manager.py --warm-up

# 重新加载内存缓存
python tools/cache_manager.py --reload

# 显示缓存统计
python tools/cache_manager.py --stats

# 清理Redis缓存
python tools/cache_manager.py --clear-redis --pattern "domain:*"
```

### 功能说明

- `--warm-up`: 从数据库重新构建宽表缓存
- `--reload`: 重新加载内存中的股票/债券缓存
- `--stats`: 显示缓存统计信息
- `--clear-redis`: 按模式删除Redis键

### 缓存统计项

- 股票缓存数量
- 债券映射数量
- 有债券股票比例
