# 数据库诊断工具使用说明

## db_inspector.py

检查数据库表结构、字段、索引、数据量

### 使用示例

```bash
# 检查单个表结构
python tools/db_inspector.py --table monitor_gp_sssj_20260421

# 按模式匹配多个表
python tools/db_inspector.py --pattern "monitor_%" --check-index

# 列出所有表
python tools/db_inspector.py --all-tables
```

### 输出说明

- 字段数、字段类型
- 索引数、索引类型
- 数据量统计
- 样本数据

## table_validator.py

验证特定表的数据完整性

### 使用示例

```bash
# 验证债券缓存表
python tools/table_validator.py --check-bond

# 检查债券到期情况
python tools/table_validator.py --check-expire

# 验证行业成分股
python tools/table_validator.py --check-industry
```

### 验证项

- 债券缓存: 总股票数、有债券比例、样本展示
- 债券到期: data_bond_ths vs data_bond_qs_jsl 对比
- 行业成分: 行业数、股票数、行业分布
