## 一、需求背景

### 当前问题
涨停分析页面的股票筛选功能（股票名称/代码输入框）只能查询**指定日期**的数据，无法跨日期查看某只股票的历史涨停记录。

### 期望功能
当使用**股票筛选框**时：
1. **跨所有日期**查询该股票的涨停记录
2. 结果按 **日期倒序 + 涨停时间倒序** 排列（最新的在前）
3. 展示中**增加日期字段**用于区分不同日期

### 默认情况不变
- 只选择日期，不输入股票名称/代码：保持现有逻辑，查询该日期所有涨停

#### URL参数变化
```javascript

/api/ztb/list?page_size=1000&date=20260422

// 股票筛选查询（跨日期）
/api/ztb/list?page_size=1000&stock_name=xxx&cross_date=1
```

#### 展示变化
- 跨日期模式：表头增加"日期"列，每行显示 `trade_date`
- 普通模式：保持现有展示，不显示日期列

### 3.2 后端逻辑修改

#### 参数定义
```python
cross_date: int = None  # 1=跨日期查询, None/0=普通查询
```

#### SQL变化

**普通查询（现有）**:
```sql
SELECT ... FROM analysis_ztb_detail_2026
WHERE trade_date = '2026-04-22'
  AND stock_name LIKE '%xxx%'
ORDER BY zt_time ASC
```

**跨日期查询（新增）**:
```sql
SELECT ... FROM analysis_ztb_detail_2026
WHERE stock_name LIKE '%xxx%'
ORDER BY trade_date DESC, zt_time DESC
```

### 3.3 排序规则

| 模式 | 排序字段 | 顺序 |
|------|----------|------|
| 普通 | zt_time | ASC (早→晚) |
| 跨日期 | trade_date DESC, zt_time DESC | 新→旧 |

```javascript
function loadZTBList() {
    const dateInput = document.getElementById('ztb-date').value;
    const stock = document.getElementById('ztb-stock').value;
    const marketFilter = document.getElementById('ztb-market').value;
    // ... 其他筛选条件
    
    // 判断是否使用跨日期筛选模式：当输入了股票名称/代码时
    const isCrossDate = stock && stock.trim() !== '';
    
    let url = `/api/ztb/list?page_size=1000`;
    
    if (isCrossDate) {
        // 跨日期模式：不传date，传cross_date=1
        url += `&cross_date=1`;
        // 股票名称必选
        url += `&stock_name=${encodeURIComponent(stock)}`;
    } else {
        // 普通模式：按日期查询
        if (dateInput) {
            url += `&date=${dateInput.replace(/-/g, '')}`;
        }
        if (stock) url += `&stock_name=${encodeURIComponent(stock)}`;
    }
    
    // ... 其他参数
    if (marketFilter && marketFilter !== 'all') url += `&market_filter=${marketFilter}`;
    // ...
}
```

#### 修改 `renderZTBList()` 函数
```javascript
function renderZTBList(items) {
    if (!items || items.length === 0) {
        document.getElementById('ztb-list').innerHTML = '<div class="empty">暂无数据</div>';
        return;
    }
    
    // 判断是否跨日期模式（根据返回的标识或数据特征）
    const isCrossDate = items.length > 0 && items[0]._cross_date === true;
    
    let html = '<table class="ztb-table"><thead><tr>';
    
    // 跨日期模式显示日期列
    if (isCrossDate) {
        html += '<th>日期</th>';
    }
    
    html += '<th>涨停时间</th><th>股票代码</th><th>股票名称</th>...';
    html += '</tr></thead><tbody>';
    
    items.forEach(item => {
        html += '<tr>';
        
        // 跨日期模式显示日期
        if (isCrossDate) {
            html += `<td>${item.trade_date || '-'}</td>`;
        }
        
        html += `<td>${item.zt_time || '-'}</td>`;
        html += `<td>${item.stock_code || '-'}</td>`;
        // ... 其他列
        
        html += '</tr>';
    });
    
    html += '</tbody></table>';
    document.getElementById('ztb-list').innerHTML = html;
}
```
