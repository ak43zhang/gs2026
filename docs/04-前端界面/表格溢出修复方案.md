# 表格内容溢出修复方案

## 问题
- 涨跌幅、债券涨跌幅等内容超出单元格边界
- 固定宽度与长内容冲突

## 解决方案

### 方案1：允许特定列换行（推荐）
涨跌幅、债券涨跌幅列允许换行，其他列保持不换行：

```css
/* 涨跌幅列允许换行 */
.results-table th:nth-child(3), 
.results-table td:nth-child(3),
.results-table th:nth-child(7), 
.results-table td:nth-child(7) {
    white-space: normal;
    word-break: break-all;
}
```

### 方案2：增加列宽 + 溢出省略
增加涨跌幅列宽度，超长内容显示省略号：

```css
/* 增加宽度 */
.results-table th:nth-child(3), .results-table td:nth-child(3) { width: 75px; }
.results-table th:nth-child(7), .results-table td:nth-child(7) { width: 85px; }

/* 溢出省略 */
.results-table td {
    overflow: hidden;
    text-overflow: ellipsis;
}
```

### 方案3：减小字体 + 调整padding
```css
.results-table {
    font-size: 12px;  /* 原13px */
}
.results-table th, .results-table td {
    padding: 8px 10px;  /* 减小padding */
}
```

## 推荐方案
**方案1 + 微调列宽**：
- 涨跌幅、债券涨跌幅允许换行
- 适当增加这两列宽度（65px→70px, 75px→80px）
- 其他列保持不换行

这样既能显示完整内容，又保持表格整齐。
