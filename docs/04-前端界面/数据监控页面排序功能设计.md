# 数据监控页面排序功能设计

> 创建时间: 2026-04-22 10:35
> 版本: 1.0

---

## 一、功能概述

数据监控页面支持对股票排行和债券排行表格进行多维度排序。

### 1.1 可排序列

| 列名 | 字段 | 排序类型 | 适用排行 |
|------|------|----------|----------|
| 涨跌幅 | `change_pct` | 数值排序 | 股票排行、债券排行 |
| 次数 | `count` | 数值排序 + 涨次排序 | 股票排行、债券排行 |
| 债券代码 | `bond_code` | 分组排序 | 股票排行 |
| 行业 | `industry` | 分组排序 | 股票排行 |

---

## 二、排序逻辑详解

### 2.1 涨跌幅排序 (`change_pct`)

**逻辑**: 按涨跌幅数值排序

```javascript
sorted.sort((a, b) => {
    const aVal = parseFloat(a.change_pct) || null;
    const bVal = parseFloat(b.change_pct) || null;
    if (aVal === null && bVal === null) return 0;
    if (aVal === null) return 1;
    if (bVal === null) return -1;
    return (aVal - bVal) * dir;
});
```

**模式**: 降序 → 升序 → 恢复原始

### 2.2 次数排序 (`count`)

**逻辑**: 先按次数排序，次数相同按涨跌幅降序

```javascript
sorted.sort((a, b) => {
    const aVal = parseInt(a.count) || 0;
    const bVal = parseInt(b.count) || 0;
    if (aVal !== bVal) return (aVal - bVal) * dir;
    // 次数相同时按涨跌幅降序
    const aPct = parseFloat(a.change_pct) || 0;
    const bPct = parseFloat(b.change_pct) || 0;
    return bPct - aPct;
});
```

### 2.3 债券代码排序 (`bond_code`) - 方案B

**逻辑**: 
1. 有债券代码的排前面，无债券代码的排后面
2. 每组内部按涨跌幅排序

**实现**:

```javascript
const hasBondCode = (item) => {
    const bc = item.bond_code;
    return bc && bc !== '-' && bc !== '未知' && bc !== null && bc !== undefined;
};

// 分离有代码和无代码的数据
const withCode = sorted.filter(hasBondCode);
const withoutCode = sorted.filter(item => !hasBondCode(item));

// 分别按涨跌幅排序
const sortByPct = (a, b) => {
    const aPct = parseFloat(a.change_pct) || -999;
    const bPct = parseFloat(b.change_pct) || -999;
    return (bPct - aPct) * dir;
};

withCode.sort(sortByPct);
withoutCode.sort(sortByPct);

// 合并：有代码的在前，无代码的在后
sorted = [...withCode, ...withoutCode];
```

**排序效果**:

```
原始数据:
| 债券代码 | 股票名称 | 涨跌幅 |
|----------|----------|--------|
| 127010   | 平安银行 | +3.2%  |  ← 有代码
| -        | 张三股份 | +8.5%  |  ← 无代码
| 123001   | 李四科技 | +5.1%  |  ← 有代码
| -        | 王五能源 | +2.0%  |  ← 无代码

点击债券代码排序后（按涨跌幅降序）:

|----------|----------|--------|
| 123001   | 李四科技 | +5.1%  |  ← 有代码组，涨跌幅最高
| 127010   | 平安银行 | +3.2%  |  ← 有代码组，涨跌幅次之
| -        | 张三股份 | +8.5%  |  ← 无代码组，涨跌幅最高
| -        | 王五能源 | +2.0%  |  ← 无代码组，涨跌幅次之
```

### 2.4 行业排序 (`industry`)

**逻辑**: 按行业出现数量分组，组内按涨跌幅排序

```javascript
// 统计各行业出现次数
const industryCounts = {};
sorted.forEach(item => {
    const ind = item.industry_name || '未知';
    industryCounts[ind] = (industryCounts[ind] || 0) + 1;
});

// 按次数排序，次数相同按行业名，组内按涨跌幅
sorted.sort((a, b) => {
    const aInd = a.industry_name || '未知';
    const bInd = b.industry_name || '未知';
    const aCount = industryCounts[aInd] || 0;
    const bCount = industryCounts[bInd] || 0;
    if (aCount !== bCount) return (aCount - bCount) * dir;
    if (aInd !== bInd) return aInd.localeCompare(bInd);
    const aPct = parseFloat(a.change_pct) || -999;
    const bPct = parseFloat(b.change_pct) || -999;
    return bPct - aPct;
});
```

## 三、交互设计

### 3.1 表头样式

```html
<th class="sortable-th" onclick="toggleRankSort('stock-ranking','change_pct')">
    涨跌幅
</th>
<th class="sortable-th sort-active" onclick="toggleRankSort('stock-ranking','count')">
    次数<span class="sort-arrow">▼</span>
</th>
```

### 3.2 状态管理

```javascript
// 排序状态存储
_rankSort[rankId] = {
    field: 'count',    // 当前排序字段
    dir: 'desc'        // 排序方向: 'desc' | 'asc'
};
```

### 3.3 切换逻辑

```javascript
function toggleRankSort(rankId, field) {
    const st = _rankSort[rankId] || { field: null, dir: 'desc' };
    
    if (st.field === field) {
        // 同一字段：降序 → 升序 → 取消
        if (st.dir === 'desc') { st.dir = 'asc'; }
        else { st.field = null; st.dir = 'desc'; }
    } else {
        // 不同字段：默认降序
        st.field = field;
        st.dir = 'desc';
    }
    
    // 重新渲染
    if (_rankRawData[rankId]) {
        renderRanking(rankId, _rankRawData[rankId]);
    }
}
```
