# 涨停行概选股 + 交叉行概选股 — 无分组展示方案

> 两个面板共用同一套展示模式切换 + 排序逻辑

---

## 一、需求

在**交叉行概选股**和**涨停行概选股**两个面板中，各自增加：
1. **展示模式下拉框**（分组展示 / 无分组展示），位于"仅显示有债券"左侧
2. **无分组展示**：去掉分组标题，所有股票放一张表，新增"命中个数"列
3. **三列可排序**：涨跌幅、债券涨跌幅、命中个数，支持降序→升序→还原三态
4. **默认排序**：命中个数降序

### 数据结构

后端返回的 `data.groups[].stocks[]` 每个stock对象包含：
```json
{
  "stock_code": "000001",
  "stock_name": "平安银行",
  "change_pct": 2.35,
  "price": 12.30,
  "bond_code": "127045",
  "bond_name": "平银转债",
  "bond_change_pct": 1.20,
  "matched_industries": ["银行"],
  "matched_concepts": ["沪深300", "融资融券"],
  "match_count": 5    // 来自group.match_count
}
```

**注意**：`match_count` 在 `group` 级别，需要在展平时注入到每个 `stock` 对象中。

#### （3）新增 renderFlatTable 函数

```javascript
function renderFlatTable(filteredGroups, container, isZtbPanel) {
    const sortState = isZtbPanel ? ztbSortState : crossSortState;
    
    // 1. 展平：将所有groups的stocks合并，注入match_count
    let allStocks = [];
    filteredGroups.forEach(group => {
        group.stocks.forEach(stock => {
            allStocks.push({ ...stock, match_count: group.match_count });
        });
    });

    // 2. 排序
    if (sortState.direction !== 'default') {
        const field = sortState.field;
        const asc = sortState.direction === 'asc';
        allStocks.sort((a, b) => {
            let va = parseFloat(a[field]) || 0;
            let vb = parseFloat(b[field]) || 0;
            return asc ? va - vb : vb - va;
        });
    } else {
        // 默认：match_count降序
        allStocks.sort((a, b) => (b.match_count || 0) - (a.match_count || 0));
    }

    // 3. 生成排序箭头
    function sortIcon(field) {
        if (sortState.field === field && sortState.direction !== 'default') {
            const arrow = sortState.direction === 'desc' ? '↓' : '↑';
            return `<span class="sort-arrow active">${arrow}</span>`;
        }
        return `<span class="sort-arrow">↕</span>`;
    }
    
    const panelFlag = isZtbPanel ? 'true' : 'false';

    // 4. 渲染单张表格
    container.innerHTML = `
        <table class="results-table">
            <thead><tr>
                <th>股票代码</th>
                <th>股票名称</th>
                <th class="sortable-header" onclick="handleFlatSort('change_pct', ${panelFlag})">
                    涨跌幅${sortIcon('change_pct')}</th>
                <th>现价</th>
                <th>债券代码</th>
                <th>债券名称</th>
                <th class="sortable-header" onclick="handleFlatSort('bond_change_pct', ${panelFlag})">
                    债券涨跌幅${sortIcon('bond_change_pct')}</th>
                <th class="sortable-header" onclick="handleFlatSort('match_count', ${panelFlag})">
                    命中个数${sortIcon('match_count')}</th>
                <th>所属行概</th>
            </tr></thead>
            <tbody>
                ${allStocks.map(stock => renderStockRow(stock)).join('')}
            </tbody>
        </table>
    `;
}
```

#### （4）排序切换函数

```javascript
function handleFlatSort(field, isZtbPanel) {
    const sortState = isZtbPanel ? ztbSortState : crossSortState;
    
    if (sortState.field === field) {
        // 同字段：desc → asc → default
        if (sortState.direction === 'desc') {
            sortState.direction = 'asc';
        } else if (sortState.direction === 'asc') {
            // 还原默认
            sortState.field = 'match_count';
            sortState.direction = 'desc';
        }
    } else {
        // 新字段，默认降序
        sortState.field = field;
        sortState.direction = 'desc';
    }
    
    // 重新渲染
    if (isZtbPanel) {
        renderResults(window.lastZtbQueryResult, 'ztbResultsSummary', 'ztbResultsContent');
    } else {
        renderResults(window.lastQueryResult);
    }
}
```

#### （5）renderStockRow — 抽取公共行渲染

将现有 `renderResults` 中的 `<tr>...</tr>` 逻辑抽成公共函数，分组和无分组共用：

```javascript
function renderStockRow(stock) {
    const changePct = parseFloat(stock.change_pct) || 0;
    const changeClass = changePct > 0 ? 'change-up' : (changePct < 0 ? 'change-down' : '');
    const changeSign = changePct > 0 ? '+' : '';
    
    const bondChangePct = parseFloat(stock.bond_change_pct) || 0;
    const bondChangeClass = bondChangePct > 0 ? 'change-up' : (bondChangePct < 0 ? 'change-down' : '');
    const bondChangeSign = bondChangePct > 0 ? '+' : '';
    const bondChangeDisplay = stock.bond_code && stock.bond_code !== '-' 
        ? `${bondChangeSign}${bondChangePct.toFixed(2)}%` : '-';
    const price = parseFloat(stock.price) || 0;
    
    // 所属行概标签（复用现有tags-grid逻辑）
    const allTags = [
        ...(stock.matched_industries || []).map(n => ({type:'industry', text:`🏭${n}`, fullText:n})),
        ...(stock.matched_concepts || []).map(n => ({type:'concept', text:`💡${n}`, fullText:n}))
    ];
    const showExpand = allTags.length > 5;
    const hiddenCount = allTags.length - 5;
    const tagsHtml = allTags.map((tag, i) => {
        const hidden = showExpand && i >= 5 ? ' hidden-tag' : '';
        return `<span class="tag-item tag-item-${tag.type}${hidden}" title="${tag.fullText}">${tag.text}</span>`;
    }).join('');
    const toggleBtn = showExpand ? 
        `<span class="expand-toggle-btn" onclick="toggleStockTags(this, '${stock.stock_code}')" title="展开/收起">+${hiddenCount}</span>` : '';
    
    // 命中个数列（无分组模式才显示，但函数中始终生成）
    const matchCountTd = stock.match_count !== undefined 
        ? `<td style="text-align:center; font-weight:bold; color:#2196F3;">${stock.match_count}</td>` : '';
    
    return `<tr>
        <td>${stock.stock_code}</td>
        <td>${stock.stock_name}</td>
        <td class="${changeClass}">${changeSign}${changePct.toFixed(2)}%</td>
        <td>${price > 0 ? price.toFixed(2) : '-'}</td>
        <td>${stock.bond_code || '-'}</td>
        <td>${stock.bond_name || '-'}</td>
        <td class="${bondChangeClass}">${bondChangeDisplay}</td>
        ${matchCountTd}
        <td><div class="tags-grid collapsed" id="tags-${stock.stock_code}">${tagsHtml}${toggleBtn}</div></td>
    </tr>`;
}
```

## 五、交互流程

```
用户选择标签 → 点击查询 → 后端返回data
                              ↓
                    检查展示模式下拉框
                    ├─ "分组展示" → renderGroupedTable（现有逻辑）
                    └─ "无分组展示" → renderFlatTable
                                       ├─ 展平groups → allStocks[]
                                       ├─ 注入match_count
                                       ├─ 按sortState排序
                                       └─ 渲染单张表格（含排序箭头）

用户点击排序表头 → handleFlatSort
                    ├─ 更新sortState（desc→asc→default循环）
                    └─ 重新调用renderResults → renderFlatTable
```
