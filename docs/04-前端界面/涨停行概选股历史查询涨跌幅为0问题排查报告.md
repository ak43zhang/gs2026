# 涨停行概选股历史查询涨跌幅为0问题排查报告

> 时间：2026-04-28 01:39  
> 问题：历史查询时涨跌幅和债券涨跌幅显示0.00%

---

## 问题根因

### 1. 调用链分析

```
前端 query API
    ↓ 调用（无date参数）
stock_picker_service.query_cross_stocks(selected_tags)
    ↓ 调用（无date参数）
query_realtime_prices(all_codes)
query_bond_realtime_prices(bond_codes)
    ↓ 使用默认日期
date = datetime.now().strftime('%Y%m%d')  # 今天=20260428
```

### 2. 问题确认

| 环节 | 问题 |
|------|------|
| 前端API | 未传递日期参数 |
| query_cross_stocks | 未接收日期参数 |
| 价格查询函数 | 使用默认日期（今天） |
| 结果 | 历史日期查询时，用今天的价格数据（可能不存在或为0） |

### 3. 数据验证

- 4月27日实时监控表 `monitor_gp_sssj_20260427`：**存在**
- 4月27日Redis数据：**存在**
- 但查询时使用今天（4月28日）的日期，导致查不到数据
