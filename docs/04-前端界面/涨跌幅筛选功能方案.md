# 涨跌幅筛选功能方案

## 需求
- 股票涨跌幅筛选：大于1%、大于2%
- 债券涨跌幅筛选：大于1%、大于2%
- 适用于交叉和涨停两个面板

## 实现方案

### 1. 前端界面：添加筛选控件

在"仅显示有债券"筛选旁边添加两个下拉框：

```html
<!-- 交叉面板筛选区 -->
<div class="filter-section">
    <label>股票涨跌幅:</label>
    <select id="crossStockChangeFilter">
        <option value="">全部</option>
        <option value="1">大于1%</option>
        <option value="2">大于2%</option>
    </select>
    
    <label>债券涨跌幅:</label>
    <select id="crossBondChangeFilter">
        <option value="">全部</option>
        <option value="1">大于1%</option>
        <option value="2">大于2%</option>
    </select>
    
    <label><input type="checkbox" id="bondFilter"> 仅显示有债券</label>
</div>

<!-- 涨停面板同理 -->
<select id="ztbStockChangeFilter">...</select>
<select id="ztbBondChangeFilter">...</select>
```

### 2. 前端逻辑：筛选过滤

在 `renderResults` 函数中增加筛选逻辑：

```javascript
// 获取筛选值
const stockChangeThreshold = parseFloat(document.getElementById('crossStockChangeFilter').value) || 0;
const bondChangeThreshold = parseFloat(document.getElementById('crossBondChangeFilter').value) || 0;

// 过滤股票
filteredStocks = stocks.filter(stock => {
    // 股票涨跌幅筛选
    if (stockChangeThreshold > 0) {
        const changePct = parseFloat(stock.change_pct) || 0;
        if (changePct <= stockChangeThreshold) return false;
    }
    // 债券涨跌幅筛选
    if (bondChangeThreshold > 0) {
        const bondChangePct = parseFloat(stock.bond_change_pct) || 0;
        if (bondChangePct <= bondChangeThreshold) return false;
    }
    return true;
});
```

### 3. 事件监听

```javascript
document.getElementById('crossStockChangeFilter').addEventListener('change', () => {
    renderResults(currentData, 'crossSummary', 'crossResultsContent');
});
document.getElementById('crossBondChangeFilter').addEventListener('change', () => {
    renderResults(currentData, 'crossSummary', 'crossResultsContent');
});
```
