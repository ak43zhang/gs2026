## 一、功能概述

用户通过搜索框（支持拼音/汉字模糊搜索）选择多个行业或概念，系统展示这些行业/概念下所有股票及对应债券，并按交叉数量分层排序。

### 核心交互流程
```
搜索框输入 → 下拉候选列表 → 点击添加行业/概念标签
    ↓
系统查询各行业/概念成分股 → 计算交叉关系
    ↓
分层展示：
  ┌─ 框1: 命中3个行业/概念的股票（涨跌幅倒排）
  ├─ 框2: 命中2个行业/概念的股票（涨跌幅倒排）
  └─ 框3: 命中1个行业/概念的股票（涨跌幅倒排）
```

**实现逻辑**:
```python
def query_cross_stocks(tags):
    # 1. 查询每个 tag 的成分股集合
    tag_stocks = {}
    for tag in tags:
        if tag.type == 'industry':
            stocks = query_industry_stocks(tag.code)
        else:
            stocks = query_concept_stocks(tag.code)
        tag_stocks[tag.name] = set(stocks)
    
    # 2. 统计每只股票命中的 tag 数量
    stock_hit_count = {}  # stock_code -> [hit_tag_names]
    for tag_name, stocks in tag_stocks.items():
        for stock in stocks:
            stock_hit_count.setdefault(stock, []).append(tag_name)
    
    # 3. 按命中数量分组
    groups = {}  # match_count -> [stocks]
    for stock, hit_tags in stock_hit_count.items():
        count = len(hit_tags)
        groups.setdefault(count, []).append((stock, hit_tags))
    
    # 4. 查询实时涨跌幅
    all_codes = list(stock_hit_count.keys())
    price_data = query_realtime_prices(all_codes)
    
    # 5. 查询债券映射
    bond_map = query_bond_mapping(all_codes)
    
    # 6. 各组内按涨跌幅倒排
    for count in sorted(groups.keys(), reverse=True):
        groups[count].sort(key=lambda x: price_data.get(x[0], {}).get('change_pct', 0), reverse=True)
    
    return groups
```

## 四、数据预热方案（宽表缓存）

### 4.1 预热时机
- **首次启动时**：服务启动后自动预热
- **定时刷新**：每 30 分钟刷新一次（行业/概念成分股变动较少）
- **手动刷新**：提供 API `/api/stock-picker/refresh-cache`

### 4.2 宽表结构

**表名**: `cache_stock_industry_concept_bond`

| 字段 | 类型 | 说明 |
|------|------|------|
| `stock_code` | VARCHAR(10) | 股票代码（主键） |
| `stock_name` | VARCHAR(20) | 股票名称 |
| `industry_codes` | JSON | 所属行业代码列表 ["881121", ...] |
| `industry_names` | JSON | 所属行业名称列表 ["半导体", ...] |
| `concept_codes` | JSON | 所属概念代码列表 ["309055", ...] |
| `concept_names` | JSON | 所属概念名称列表 ["6G概念", ...] |
| `bond_code` | VARCHAR(10) | 可转债代码，无则为 null |
| `bond_name` | VARCHAR(50) | 可转债名称，无则为 null |
| `update_time` | DATETIME | 更新时间 |

### 4.3 预热逻辑

```python
def warm_up_cache():
    """预热宽表缓存"""
    # 1. 加载所有股票的行业归属
    industry_stocks = {}
    for row in query('SELECT stock_code, code as industry_code, name as industry_name FROM data_industry_code_component_ths'):
        industry_stocks.setdefault(row.stock_code, {'codes': [], 'names': []})
        industry_stocks[row.stock_code]['codes'].append(row.industry_code)
        industry_stocks[row.stock_code]['names'].append(row.industry_name)
    
    # 2. 加载所有股票的概念归属
    concept_stocks = {}
    for row in query('SELECT stock_code, index_code as concept_code FROM data_gnzscfxx_ths'):
        # 需要关联 ths_gn_names 获取概念名称
        concept_name = get_concept_name(row.concept_code)
        concept_stocks.setdefault(row.stock_code, {'codes': [], 'names': []})
        concept_stocks[row.stock_code]['codes'].append(row.concept_code)
        concept_stocks[row.stock_code]['names'].append(concept_name)
    
    # 3. 加载债券映射
    bond_map = {}
    for row in query('SELECT 正股代码, 债券代码, 债券名称 FROM data_bond_ths'):
        bond_map[row.正股代码] = {'code': row.债券代码, 'name': row.债券名称}
    
    # 4. 合并写入宽表
    all_stocks = set(industry_stocks.keys()) | set(concept_stocks.keys())
    for stock_code in all_stocks:
        record = {
            'stock_code': stock_code,
            'stock_name': get_stock_name(stock_code),
            'industry_codes': industry_stocks.get(stock_code, {}).get('codes', []),
            'industry_names': industry_stocks.get(stock_code, {}).get('names', []),
            'concept_codes': concept_stocks.get(stock_code, {}).get('codes', []),
            'concept_names': concept_stocks.get(stock_code, {}).get('names', []),
            'bond_code': bond_map.get(stock_code, {}).get('code'),
            'bond_name': bond_map.get(stock_code, {}).get('name'),
        }
        upsert_cache_table(record)
```

### 4.4 内存缓存

```python
# 服务启动时加载到内存
_stock_cache = {}  # stock_code -> {industries, concepts, bond}

class StockCache:
    @staticmethod
    def load():
        """从宽表加载到内存"""
        for row in query('SELECT * FROM cache_stock_industry_concept_bond'):
            _stock_cache[row.stock_code] = {
                'stock_name': row.stock_name,
                'industries': set(row.industry_names),  # 用集合方便交集计算
                'concepts': set(row.concept_names),
                'bond_code': row.bond_code,
                'bond_name': row.bond_name,
            }
    
    @staticmethod
    def query_cross_stocks(selected_tags):
        """基于内存缓存查询交叉股票"""
        selected_set = set(selected_tags)
        results = {}
        
        for stock_code, data in _stock_cache.items():
            # 计算该股票命中的标签
            all_tags = data['industries'] | data['concepts']
            matched = selected_set & all_tags
            
            if matched:
                match_count = len(matched)
                results.setdefault(match_count, []).append({
                    'stock_code': stock_code,
                    'stock_name': data['stock_name'],
                    'bond_code': data['bond_code'] or '-',
                    'bond_name': data['bond_name'] or '-',
                    'matched_tags': list(matched),
                    'matched_tags_display': '+'.join(sorted(matched)),
                })
        
        return results
```

### 4.5 性能对比

| 方案 | 首次查询耗时 | 后续查询 | 实现复杂度 |
|------|-------------|----------|-----------|
| 实时多表JOIN | 500-1000ms | 相同 | 低 |
| 宽表缓存 | 50-100ms | 相同 | 中 |
| **内存缓存** | **5-10ms** | **<1ms** | **中** |

**推荐方案**：内存缓存 + 宽表持久化

## 五、拼音搜索实现

### 方案：使用 `pypinyin` 库

```python
from pypinyin import lazy_pinyin, Style

class PinyinSearcher:
    def __init__(self):
        self.items = []  # [{name, code, type, pinyin_full, pinyin_initials}]
    
    def add(self, name, code, item_type):
        pinyin_full = ''.join(lazy_pinyin(name))       # "bandaoti"
        pinyin_initials = ''.join(lazy_pinyin(name, style=Style.FIRST_LETTER))  # "bdt"
        self.items.append({
            'name': name, 'code': code, 'type': item_type,
            'pinyin_full': pinyin_full,
            'pinyin_initials': pinyin_initials,
            'name_lower': name.lower()
        })
    
    def search(self, query, limit=20):
        query = query.lower().strip()
        results = []
        for item in self.items:
            if (query in item['name_lower'] or 
                query in item['pinyin_full'] or 
                query in item['pinyin_initials']):
                results.append(item)
        return results[:limit]
```

## 七、性能考虑

| 操作 | 数据量 | 策略 |
|------|--------|------|
| 搜索 | 465条名称（90行业+375概念） | 内存搜索，毫秒级 |
| 成分股查询 | 单行业平均61只，单概念平均180只 | MySQL查询，缓存到内存 |
| 涨跌幅查询 | 按需批量查询 | 从 `monitor_gp_sssj_{date}` 最新时间戳 |
| 债券映射 | 919条 | 启动时加载到内存 |

### 缓存策略
- 行业/概念名称：启动时加载，常驻内存
- 成分股数据：首次查询后缓存（TTL 1小时）
- 债券映射：启动时加载，常驻内存
- 涨跌幅：每次实时查询，不缓存
