# 涨停行概选股 - 所属行概展示优化方案

> 问题: 所属行概列表太长，占用过多空间
> 目标: 优化展示，使其更紧凑

---

## 当前问题

- 概念标签过多时，列表很长
- 每个标签独占一行，浪费空间
- 用户需要滚动才能看到股票数据

---

## 优化方案

### 方案A: 标签云/紧凑布局（推荐）

将垂直列表改为水平流式布局，多行展示：

```
所属行概: 💡5G 💡AI应用 💡AI智能体 💡DeepSeek概念 💡F5G概念 💡一带一路 
          💡东数西算 💡人工智能 💡低空经济 💡储能 💡充电桩 💡光伏概念
          💡光纤概念 💡CPO 💡华为概念 ... (+5)
```

**实现方式:**
- 使用 `display: flex; flex-wrap: wrap;` 布局
- 标签横向排列，自动换行
- 限制最大行数（如2行），超出显示"+N"

**优点:**
- 空间利用率高
- 视觉紧凑
- 实施简单

---

### 方案B: 折叠/展开模式

默认只显示前N个，点击展开显示全部：

```
所属行概: 💡5G 💡AI应用 💡AI智能体 💡DeepSeek概念 💡F5G概念 [+15]
```

点击 [+15] 后展开全部。

**实现方式:**
- 默认渲染前5个标签
- 添加"展开/收起"按钮
- 使用 CSS max-height + transition 动画

**优点:**
- 默认状态非常紧凑
- 用户可控，想看详情可以展开

---

### 方案C: 分组统计模式

按行业/概念分组，显示数量：

```
所属行概: 🏭半导体(2) 🏭电力(1) 💡AI应用(3) 💡新能源(2) [查看详情]
```

点击 [查看详情] 弹出完整列表。

**优点:**
- 信息密度最高
- 适合快速浏览

---

## 推荐方案: 方案A + 方案B 结合

### 设计细节

1. **默认状态**: 显示前2行标签，超出显示 "+N"
2. **布局**: Flex 流式布局，标签紧凑排列
3. **交互**: 点击 "+N" 展开全部，再次点击收起

### 样式设计

```css
/* 容器 */
.tags-container {
    display: flex;
    flex-wrap: wrap;
    gap: 4px 8px;  /* 行间距4px，列间距8px */
    max-height: 48px;  /* 约2行高度 */
    overflow: hidden;
    position: relative;
}

/* 展开状态 */
.tags-container.expanded {
    max-height: none;
}

/* 标签样式 */
.tag {
    display: inline-flex;
    align-items: center;
    padding: 2px 6px;
    font-size: 12px;
    border-radius: 3px;
    white-space: nowrap;
}

.tag-industry {
    background: #E3F2FD;
    color: #1565C0;
}

.tag-concept {
    background: #FFF3E0;
    color: #E65100;
}

/* 展开按钮 */
.expand-btn {
    color: #2196F3;
    cursor: pointer;
    font-size: 12px;
}
```

### HTML结构

```html
<div class="tags-wrapper">
    <span class="label">所属行概:</span>
    <div class="tags-container" id="tagsContainer">
        <span class="tag tag-industry">🏭半导体</span>
        <span class="tag tag-concept">💡5G</span>
        <span class="tag tag-concept">💡AI应用</span>
        <!-- 更多标签... -->
    </div>
    <span class="expand-btn" onclick="toggleTags()">+15</span>
</div>
```

### JavaScript交互

```javascript
function toggleTags() {
    const container = document.getElementById('tagsContainer');
    const btn = document.querySelector('.expand-btn');
    
    if (container.classList.contains('expanded')) {
        container.classList.remove('expanded');
        btn.textContent = `+${hiddenCount}`;
    } else {
        container.classList.add('expanded');
        btn.textContent = '收起';
    }
}

// 初始化时计算是否显示展开按钮
function initTags() {
    const container = document.getElementById('tagsContainer');
    const tags = container.querySelectorAll('.tag');
    
    // 如果标签高度超过2行，显示展开按钮
    if (container.scrollHeight > 48) {
        let visibleCount = 0;
        let currentHeight = 0;
        
        for (const tag of tags) {
            currentHeight += tag.offsetHeight;
            if (currentHeight <= 48) {
                visibleCount++;
            } else {
                break;
            }
        }
        
        const hiddenCount = tags.length - visibleCount;
        document.querySelector('.expand-btn').textContent = `+${hiddenCount}`;
    } else {
        document.querySelector('.expand-btn').style.display = 'none';
    }
}
```

## 对比效果

| 方案 | 默认占用高度 | 信息完整度 | 实施难度 |
|------|-------------|-----------|---------|
| 当前 | 很高（N行） | 完整 | - |
| 方案A | 低（2行） | 完整（展开后） | 低 |
| 方案B | 很低（1行） | 部分 | 中 |
| 方案C | 很低（1行） | 摘要 | 中 |
| **推荐** | **低（2行）** | **完整** | **低** |
