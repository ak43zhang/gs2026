## 问题列表

### 问题1: 页面标题不随Tab切换更新

**现象**: 点击"涨停行概选股"Tab后，浏览器标签页标题仍显示"交叉行概选股"

**原因**: HTML的`<title>`标签是静态文本，切换Tab时未更新

**修复方案**: 在`switchTab`函数中添加`document.title`更新

```javascript
// 切换Tab时更新页面标题
document.title = tab === 'cross' 
    ? '交叉行概选股 - 智能选股 - GS2026'
    : '涨停行概选股 - 智能选股 - GS2026';
```

### 问题2: "仅显示有债券"筛选失效

**现象**: 取消勾选"仅显示有债券"后，结果仍只显示有债券的股票

**原因**: 在`renderResults`函数中，当`onlyWithBond`为false时，筛选逻辑未正确处理

**修复方案**: 修改筛选逻辑，确保`onlyWithBond`为false时显示所有股票

```javascript
// 修复前的问题代码
if (onlyWithBond) {
    // 筛选逻辑...
    withBondCount = totalStocks; // 这里有问题
}

// 修复后
if (onlyWithBond) {
    // 筛选逻辑...
}
// withBondCount 应该在筛选前计算
```

### 问题3: 展开按钮重复显示

**现象**: 所属行概标签显示两个"+N"展开按钮

**原因**: 代码中生成了两个展开按钮
- 第一个在`visibleTags`后（第4个位置）
- 第二个在`hiddenTagsHtml`最后（作为收起按钮）

**修复方案**: 只保留一个展开按钮，放在visibleTags后面，hiddenTags中只放标签和收起按钮

```javascript
// 修复前
const visibleTags = showExpand ? allTags.slice(0, 4) : allTags;
const expandBtnHtml = showExpand ? `<span class="expand-toggle-btn">+${hiddenCount}</span>` : '';
const hiddenTagsHtml = showExpand ? allTags.slice(4).map(...).join('') : '';

// 修复后
const visibleTags = showExpand ? allTags.slice(0, 5) : allTags;  // 显示5个
const expandBtnHtml = showExpand ? `<span class="expand-toggle-btn">+${hiddenCount}</span>` : '';
// hiddenTagsHtml 中不再包含展开按钮，只包含标签和收起按钮
```
