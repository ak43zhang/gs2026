# 表格列宽调整方案

## 问题
- 命中个数列太宽（数字不需要大空间）
- 命中行概列太窄（标签显示拥挤，表格溢出）

## 解决方案

### 保持前8列不变
```css
/* 前8列保持原固定宽度 */
.results-table th:nth-child(1), .results-table td:nth-child(1) { width: 75px; }
.results-table th:nth-child(2), .results-table td:nth-child(2) { width: 85px; }
.results-table th:nth-child(3), .results-table td:nth-child(3) { width: 65px; text-align: right; }
.results-table th:nth-child(4), .results-table td:nth-child(4) { width: 60px; text-align: right; }
.results-table th:nth-child(5), .results-table td:nth-child(5) { width: 75px; }
.results-table th:nth-child(6), .results-table td:nth-child(6) { width: 85px; }
.results-table th:nth-child(7), .results-table td:nth-child(7) { width: 75px; text-align: right; }
.results-table th:nth-child(8), .results-table td:nth-child(8) { width: 110px; }
```

### 命中个数列收窄
```css
/* 命中个数（第9列）- 固定窄宽度 */
.results-table th:nth-child(9), 
.results-table td:nth-child(9) { 
    width: 50px;  /* 收窄 */
    text-align: center;
    white-space: nowrap;
}
```

### 命中行概列加宽
```css
/* 命中行概（第10列）- 占据剩余空间 */
.results-table th:nth-child(10), 
.results-table td:nth-child(10) { 
    width: auto;
    min-width: 500px;  /* 加宽 */
    white-space: normal;
    text-align: center;
}
```

### 表格布局
```css
.results-table {
    table-layout: fixed;  /* 固定布局 */
    width: 100%;
}
```

## 效果
- 前8列保持舒适宽度
- 命中个数列收窄（50px）
- 命中行概列占据剩余空间（min-width: 500px）
- 标签有足够空间展示，不溢出
