## 二、排序变化（无分组模式）

| 可排序列 | 排序方式 | 默认 |
|---------|---------|------|
| 涨跌幅 | 降序→升序→还原 | ❌ |
| 债券涨跌幅 | 降序→升序→还原 | ❌ |
| **行业（新增）** | **按行业名称字母排序：A→Z / Z→A / 还原** | ❌ |
| 命中个数 | 降序→升序→还原 | ✅ 默认降序 |

行业排序规则：字符串排序（中文按拼音排序），支持三态切换。

## 三、修改清单

### 3.1 后端：`stock_picker_service.py`（1处）

`query_cross_stocks()` 中 `stock_result` 新增 `industry_name` 字段：

**数据来源**：`_stock_cache[stock_code]['industries']` — 该股票在 `data_industry_code_component_ths` 表中的所有行业名称，与选中标签无关。

#### ⑤ handleFlatSort 排序逻辑

行业字段为字符串，排序方式改为字符串比较：

```javascript
allStocks.sort((a, b) => {
    if (field === 'industry_name') {
        // 字符串排序（中文按localeCompare）
        let va = a[field] || '';
        let vb = b[field] || '';
        return asc ? va.localeCompare(vb, 'zh-CN') : vb.localeCompare(va, 'zh-CN');
    } else {
        // 数值排序
        let va = parseFloat(a[field]) || 0;
        let vb = parseFloat(b[field]) || 0;
        return asc ? va - vb : vb - va;
    }
});
```
