# 表格样式优化方案

## 改动内容

### 1. 命中行概列头居中
```css
/* 命中行概表头居中 */
.results-table th:nth-child(9):not(.sortable-header),
.results-table th:nth-child(10) {
    text-align: center;
}
```

### 2. 增加表格边框线
```css
/* 表格添加边框线 */
.results-table {
    border: 1px solid #e0e0e0;
}

.results-table th,
.results-table td {
    border-right: 1px solid #e0e0e0;
    border-bottom: 1px solid #e0e0e0;
}

.results-table th:last-child,
.results-table td:last-child {
    border-right: none;
}

.results-table tr:last-child td {
    border-bottom: none;
}
```

## 效果
- 命中行概表头文字居中显示
- 表格有清晰的边框线，行列分隔明确
- 整体更整洁美观
