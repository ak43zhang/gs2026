# 涨跌幅筛选选项扩充方案

## 当前状态
已有筛选下拉框：
- 股票涨跌幅：不限、涨(>0%)
- 债券涨跌幅：不限、涨(>0%)

## 修改内容

在现有下拉框中增加两个选项：
- 涨>1%
- 涨>2%

## 具体改动

### 股票涨跌幅筛选
```html
<select id="stockChangeFilter">
    <option value="">不限</option>
    <option value="0">涨(>0%)</option>
    <option value="1">涨>1%</option>  <!-- 新增 -->
    <option value="2">涨>2%</option>  <!-- 新增 -->
</select>
```

### 债券涨跌幅筛选
```html
<select id="bondChangeFilter">
    <option value="">不限</option>
    <option value="0">涨(>0%)</option>
    <option value="1">涨>1%</option>  <!-- 新增 -->
    <option value="2">涨>2%</option>  <!-- 新增 -->
</select>
```

## 筛选逻辑

前端根据选择的值过滤：
- value="" : 显示全部
- value="0" : change_pct > 0
- value="1" : change_pct > 1  (新增)
- value="2" : change_pct > 2  (新增)

## 修改文件

| 文件 | 改动 |
|------|------|
| `stock_picker.html` | 在现有4个下拉框（交叉2个+涨停2个）中各增加2个选项 |

- 债券涨跌幅筛选：不限 / 涨(>0%) / 涨>1% / 涨>2%
