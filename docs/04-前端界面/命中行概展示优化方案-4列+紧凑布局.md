# 命中行概展示优化方案

> 目标：固定4列网格 + 保留折叠 + 前段字段紧凑 + 命中行概宽松

---

## 一、当前问题

1. 命中行概列宽度不够，标签显示拥挤
2. 股票代码、股票名称等字段占用空间过大
3. 需要固定4列展示，超出折叠

---

## 二、修改方案

### 2.1 CSS调整：表格列宽分配

```css
/* 表格固定列宽 */
.results-table {
    table-layout: fixed;  /* 已添加 */
}

/* 前段字段：紧凑不换行 */
.results-table th:nth-child(1),  /* 股票代码 */
.results-table td:nth-child(1) {
    width: 70px;
    white-space: nowrap;
}

.results-table th:nth-child(2),  /* 股票名称 */
.results-table td:nth-child(2) {
    width: 80px;
    white-space: nowrap;
}

.results-table th:nth-child(3),  /* 涨跌幅 */
.results-table td:nth-child(3) {
    width: 60px;
    white-space: nowrap;
}

.results-table th:nth-child(4),  /* 现价 */
.results-table td:nth-child(4) {
    width: 60px;
    white-space: nowrap;
}

.results-table th:nth-child(5),  /* 债券代码 */
.results-table td:nth-child(5) {
    width: 70px;
    white-space: nowrap;
}

.results-table th:nth-child(6),  /* 债券名称 */
.results-table td:nth-child(6) {
    width: 80px;
    white-space: nowrap;
}

.results-table th:nth-child(7),  /* 债券涨跌幅 */
.results-table td:nth-child(7) {
    width: 70px;
    white-space: nowrap;
}

.results-table th:nth-child(8),  /* 行业 */
.results-table td:nth-child(8) {
    width: 100px;
    white-space: nowrap;
}

/* 命中行概：宽松，固定4列 */
.results-table th:nth-child(9),  /* 命中行概（分组模式第9列） */
.results-table td:nth-child(9) {
    width: auto;  /* 剩余空间 */
    min-width: 300px;
}

/* 无分组模式：命中个数列 */
.results-table th:nth-child(9),  /* 命中个数 */
.results-table td:nth-child(9) {
    width: 60px;
    white-space: nowrap;
}

.results-table th:nth-child(10), /* 命中行概（无分组第10列） */
.results-table td:nth-child(10) {
    width: auto;
    min-width: 300px;
}
```

### 2.2 标签网格：固定4列

```css
/* 标签网格固定4列 */
.tags-grid {
    display: grid;
    grid-template-columns: repeat(4, 1fr);  /* 固定4列 */
    gap: 4px;
}

/* 标签样式微调 */
.tag-item {
    font-size: 11px;
    padding: 2px 4px;
    overflow: hidden;
    text-overflow: ellipsis;
    white-space: nowrap;
}
```

### 2.3 恢复折叠功能

恢复之前的折叠逻辑：
- 默认显示4个标签（`visibleCount = 4`）
- 超出显示 `+N` 按钮
- 点击展开显示全部

### 2.4 修改文件

| 文件 | 改动 |
|------|------|
| `stock_picker.html` | ① CSS增加列宽定义 ② 恢复折叠逻辑（visibleCount=4） ③ 标签网格改为4列 |
