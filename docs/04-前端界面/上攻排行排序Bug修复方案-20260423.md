# 上攻排行债券代码排序Bug修复方案

> 日期: 2026-04-23
> 问题: 点击时间轴某个时间后，再点击债券代码排序，返回的数据是最新时间而不是时间轴指定时间的数据

---

## 问题分析

### 调用链路

```
时间轴点击 → loadDataAtTime(timeStr)
    → loadStockRankingAtTime(timeStr)    ← 股票排行
    → loadRankingAtTime('bond', timeStr) ← 债券排行
```

### 问题根因

`loadStockRankingAtTime` 函数获取数据后，**没有保存到 `_rankRawData['stock-ranking']`**：

```javascript
// 第744行 - loadStockRankingAtTime (有BUG)
async function loadStockRankingAtTime(timeStr) {
    ...
    const r = await (await fetch(url)).json();
    if (r.success) renderRanking('stock-ranking', r.data);  // ❌ 没有保存 _rankRawData
}

// 第756行 - loadRankingAtTime (正确)
async function loadRankingAtTime(type, timeStr) {
    ...
    const r = await (await fetch(url)).json();
    if (r.success) {
        const rankId = type + '-ranking';
        if (_rankRawData.hasOwnProperty(rankId)) _rankRawData[rankId] = r.data;  // ✅ 保存了
        renderRanking(rankId, r.data);
    }
}
```

### 排序时的行为

```javascript
function toggleRankSort(rankId, field) {
    ...
    if (_rankRawData[rankId]) {
        renderRanking(rankId, _rankRawData[rankId]);  // 使用 _rankRawData 中的旧数据！
    }
}
```

**结果**：点击时间轴后，股票排行显示了指定时间的数据（来自API返回），但 `_rankRawData['stock-ranking']` 还是之前实时模式的旧数据。点击排序时使用旧数据渲染，导致数据回到最新时间的状态。

---

## 修复方案

### 修改 `loadStockRankingAtTime` 函数

在渲染前保存原始数据到 `_rankRawData`：

```javascript
// 修复前
async function loadStockRankingAtTime(timeStr) {
    showRefreshIndicator('stock');
    try {
        let url = '/api/monitor/attack-ranking/stock?limit=60';
        if (timeStr) url += `&time=${timeStr}`;
        const d = getSelectedDate();
        if (d) url += `&date=${d}`;
        const r = await (await fetch(url)).json();
        if (r.success) renderRanking('stock-ranking', r.data);  // ❌
    } catch(e) {}
}

// 修复后
async function loadStockRankingAtTime(timeStr) {
    showRefreshIndicator('stock');
    try {
        let url = '/api/monitor/attack-ranking/stock?limit=60';
        if (timeStr) url += `&time=${timeStr}`;
        const d = getSelectedDate();
        if (d) url += `&date=${d}`;
        const r = await (await fetch(url)).json();
        if (r.success) {
            _rankRawData['stock-ranking'] = r.data;  // ✅ 保存原始数据
            renderRanking('stock-ranking', r.data);
        }
    } catch(e) {}
}
```

### 修改文件

- `src/gs2026/dashboard2/templates/monitor.html` (第744-752行)

### 影响范围

- 仅修改一行代码，不影响其他功能
- 所有排行榜的排序行为保持一致

1. 点击时间轴某个时间点（如10:00:00）
2. 确认股票排行数据更新为10:00:00的数据
3. 点击"债券代码"排序
4. 确认排序后数据仍然是10:00:00的数据（而非最新数据）
5. 切换回实时模式，确认排序正常
