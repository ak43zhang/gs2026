## 文档信息
- 创建时间：2026-04-28
- 文档版本：v1.0

## 一、需求背景

### 现状问题
1. **行业上攻排行**（3列：排名、代码、名称+次数）占据1/3宽度，但内容很少，空间浪费
2. **股票上攻排行**需要新增"主力净额"字段，但当前宽度不够
3. 三个面板等宽（各33.3%），股票排行列多（9列）显示拥挤

### 解决方案
- 调整grid比例：股票:债券:行业 = 3:2:1
- 股票排行增加"主力净额"字段
- 主力净额跟随涨跌幅，从monitor_gp_sssj实时查询

---

## 二、数据流分析

### 当前涨跌幅获取逻辑

```
data_service.get_stock_ranking()
    → 从 Redis 获取: code, name, count
    → 或从 MySQL rank_stock 表获取: code, name, count, date

_enrich_change_pct()
    → 从 Redis 获取最新时间戳
    → 从 monitor_gp_sssj_{date} 表批量查询 change_pct
    → 填充到每个 stock 对象

_enrich_stock_data()
    → 补充债券代码、债券名称、行业信息
```

### 关键发现
**涨跌幅 `change_pct` 不是从 Redis 获取的，而是实时从 `monitor_gp_sssj` 表查询的！**

每次 API 调用：
1. 从 Redis 获取基础排行数据（code, name, count）— **~5ms**
2. 从 MySQL `monitor_gp_sssj` 批量查询涨跌幅 — **~20ms（1次查询60条）**

---

## 三、设计方案

### 3.1 核心设计原则

**主力净额跟随涨跌幅，采用完全相同的获取逻辑**

理由：
1. 数据一致性：都来自同一表（monitor_gp_sssj），同一时间点
2. 代码简洁：只需修改 `_enrich_change_pct` 函数，同时查询两个字段
3. 无需修改 Redis：减少复杂度
4. 性能可接受：仅增加约5ms

### 3.2 修改文件清单

| 序号 | 文件 | 修改内容 | 优先级 |
|------|------|----------|--------|
| 1 | `monitor_stock.py` | `calculate_top30_v3` 增加主力净额字段映射 | P0 |
| 2 | `monitor.py` | 新增 `_enrich_change_pct_and_main_net` 函数 | P0 |
| 3 | `monitor.py` | 修改 API 调用处使用新函数 | P0 |
| 4 | `monitor.html` | 增加主力净额列显示 | P0 |

#### 4.1.2 monitor.py - 新增 _enrich_change_pct_and_main_net 函数

**目标**：批量获取涨跌幅和主力净额

```python
def _enrich_change_pct_and_main_net(stocks: list, date: str, time_str: str = None) -> list:
    """
    为股票数据添加涨跌幅和主力净额（批量优化版）
    从 monitor_gp_sssj 表批量获取 change_pct 和 main_net_amount
    【替代原 _enrich_change_pct 函数】
    """
    if not stocks:
        return stocks

    try:
        from gs2026.utils import redis_util
        client = redis_util._get_redis_client()

        # 确定查询时间
        if time_str:
            query_time = time_str
        else:
            sssj_table = f"monitor_gp_sssj_{date}"
            ts_key = f"{sssj_table}:timestamps"
            latest_ts = client.lindex(ts_key, 0)
            if latest_ts:
                query_time = latest_ts.decode('utf-8') if isinstance(latest_ts, bytes) else latest_ts
            else:
                for stock in stocks:
                    stock['change_pct'] = '-'
                    stock['main_net_amount'] = 0
                return stocks

        # 提取所有股票代码
        stock_codes = [s['code'].zfill(6) for s in stocks if s.get('code')]

        # 【修改】批量获取涨跌幅和主力净额（1次查询）
        change_pct_map, main_net_map = _get_change_pct_and_main_net_batch(date, query_time, stock_codes)

        # 填充数据
        for stock in stocks:
            code = stock.get('code', '').zfill(6)
            
            # 涨跌幅
            change_pct = change_pct_map.get(code)
            stock['change_pct'] = change_pct if change_pct is not None else '-'
            
            # 【新增】主力净额
            main_net = main_net_map.get(code)
            stock['main_net_amount'] = main_net if main_net is not None else 0

        return stocks

    except Exception as e:
        print(f"添加涨跌幅和主力净额失败: {e}")
        for stock in stocks:
            stock['change_pct'] = '-'
            stock['main_net_amount'] = 0
        return stocks


def _get_change_pct_and_main_net_batch(date: str, time_str: str, stock_codes: list) -> tuple:
    """
    批量获取涨跌幅和主力净额
    【新增函数】
    返回: (change_pct_map, main_net_map)
    """
    import pandas as pd
    from gs2026.utils.db_util import get_db_engine
    engine = get_db_engine()
    
    codes_str = ','.join([f"'{c}'" for c in stock_codes])
    table_name = f"monitor_gp_sssj_{date}"
    
    # 【修改】同时查询 change_pct 和 main_net_amount
    query = f"""
        SELECT stock_code, change_pct, main_net_amount
        FROM {table_name}
        WHERE time = '{time_str}' AND stock_code IN ({codes_str})
    """
    
    change_pct_map = {}
    main_net_map = {}
    
    try:
        with engine.connect() as conn:
            df = pd.read_sql(query, conn)
            for _, row in df.iterrows():
                code = str(row['stock_code']).zfill(6)
                # 涨跌幅
                if row['change_pct'] is not None:
                    change_pct_map[code] = float(row['change_pct'])
                # 主力净额
                if row['main_net_amount'] is not None:
                    main_net_map[code] = float(row['main_net_amount'])
                else:
                    main_net_map[code] = 0
    except Exception as e:
        print(f"批量查询涨跌幅和主力净额失败: {e}")
    
    return change_pct_map, main_net_map
```
