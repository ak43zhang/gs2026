# 涨停行概选股 - 所属行概展示优化方案 v3

> 问题: 标签展示不美观，文字过长、间距不够、收起按钮位置不对

---

## 当前问题分析

1. **标签文字过长**: "光伏建筑一体化"、"央企国企改革"等长文字标签占用过多空间
2. **间距问题**: 标签之间间距太小，显得拥挤
3. **收起按钮位置**: 单独一行显示"收起"，浪费空间
4. **对齐问题**: 标签没有对齐，视觉上不整齐

---

## 优化方案

### 方案A: 紧凑标签 + 悬浮提示（推荐）

**设计思路:**
- 限制标签最大宽度，过长显示省略号
- 鼠标悬浮显示完整名称
- 标签紧凑排列，但保持可读性
- 收起按钮放在最后一格

**效果:**
```
所属行概: 🏭其他社... 💡光伏建... 💡光伏概念 💡国企改革 💡土壤修复 +8
          💡央企国... 💡房屋检测 💡核污染防... 💡核电 💡水利
          💡沪股通 💡碳中和 💡碳交易 💡雄安新区 💡雅下水电... [收起]
```

**优点:**
- 整齐对齐
- 节省空间
- 信息完整（悬浮可看）
- 视觉舒适

---

### 方案B: 分类分组展示

**设计思路:**
- 行业标签和概念标签分开显示
- 行业标签优先，显示在左侧
- 概念标签显示在右侧

**效果:**
```
所属行概:
  行业: 🏭其他社会服务
  概念: 💡光伏建筑... 💡光伏概念 💡国企改革 💡土壤修复 💡央企国... +10
```

**优点:**
- 信息层次清晰
- 行业信息突出

---

### 方案C: 标签云 + 点击展开

**设计思路:**
- 使用流式布局（flex-wrap）
- 标签大小根据内容自适应
- 默认显示2行，超出折叠

**效果:**
```
所属行概: 🏭其他社会服务 💡光伏建筑一体化 💡光伏概念 💡国企改革 💡土壤修复
          💡央企国企改革 💡房屋检测 💡核污染防治 💡核电 💡水利 (+8)
```

## 推荐方案: 方案A 改进版

### 设计细节

1. **标签样式优化:**
   - 最大宽度: 80px
   - 文字过长显示省略号
   - 鼠标悬浮显示完整名称（title属性）
   - 字体大小: 11px
   - 内边距: 3px 6px

2. **布局优化:**
   - 使用固定5列网格
   - 列间距: 6px
   - 行间距: 4px
   - 最后一格放"+N"或"收起"按钮

3. **颜色优化:**
   - 行业标签: 蓝色系（#E3F2FD背景，#1565C0文字）
   - 概念标签: 橙色系（#FFF3E0背景，#E65100文字）
   - 展开按钮: 蓝色链接样式

### JavaScript修改

```javascript
// 生成标签HTML
const allTags = [
    ...(stock.matched_industries || []).map(name => 
        ({type: 'industry', text: `🏭${name}`, fullText: name})
    ),
    ...(stock.matched_concepts || []).map(name => 
        ({type: 'concept', text: `💡${name}`, fullText: name})
    )
];

const totalTags = allTags.length;
const showExpand = totalTags > 5;
const visibleTags = showExpand ? allTags.slice(0, 4) : allTags; // 前4个+展开按钮
const hiddenCount = totalTags - 4;

const tagListHtml = visibleTags.map(tag => 
    `<span class="tag-item tag-item-${tag.type}" title="${tag.fullText}">${tag.text}</span>`
).join('');

const expandBtnHtml = showExpand ? 
    `<span class="expand-toggle-btn" onclick="toggleStockTags(this, '${stock.stock_code}')" title="点击展开全部">+${hiddenCount}</span>` : '';

const hiddenTagsHtml = showExpand ? 
    allTags.slice(4).map((tag, idx) => {
        const isLast = idx === allTags.slice(4).length - 1;
        return `<span class="tag-item tag-item-${tag.type} hidden-tag" title="${tag.fullText}">${tag.text}</span>${isLast ? `<span class="expand-toggle-btn" onclick="toggleStockTags(this, '${stock.stock_code}')" title="点击收起">收起</span>` : ''}`;
    }).join('') : '';
```

### 改进点

| 方面 | 优化前 | 优化后 |
|------|--------|--------|
| 标签宽度 | 自适应，可能过长 | 固定最大宽度，省略号截断 |
| 悬浮提示 | 无 | 有（显示完整名称） |
| 收起按钮 | 单独一行 | 放在最后一格 |
| 视觉对齐 | 不整齐 | 5列网格对齐 |
| 边框 | 无 | 细边框，更精致 |

## 备选方案: 流式布局（更宽松）

如果用户觉得网格太紧凑，可以使用流式布局：

```css
.tags-grid {
    display: flex;
    flex-wrap: wrap;
    gap: 4px 6px;
    max-height: 28px;  /* 约1行高度 */
    overflow: hidden;
}

.tags-grid.expanded {
    max-height: none;
}

.tag-item {
    flex: 0 0 auto;
    max-width: 100px;
}
```

**效果:**
- 标签自然排列，不强制5列
- 更宽松，但可能不够整齐
