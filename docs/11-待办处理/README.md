## 问题分类

### 按优先级
- 🔴 **高优先级**: 立即处理，影响功能使用
- 🟡 **中优先级**: 近期处理，影响用户体验
- 🟢 **低优先级**: 按需处理，功能增强

### 按类型
- 功能缺陷
- 性能问题
- 技术债务
- 功能需求

### 按状态
- 🔴 未解决
- 🟡 处理中
- ✅ 已解决
- ⚠️ 待验证
- 🔍 待调查

## 问题提交流程

1. 发现问题后，首先记录在 [项目待办问题汇总.md](项目待办问题汇总.md)
2. 如果是已知但待验证的问题，添加到 [已知问题清单.md](已知问题清单.md)
3. StepClaw 工作站处理的问题，记录在 [StepClaw工作站记录.md](StepClaw工作站记录.md)
4. 问题解决后，更新状态并添加解决记录
