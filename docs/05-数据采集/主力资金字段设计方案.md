## 2. 主力资金计算方案

### 2.1 核心思路

**无盘口数据时的间接估算方法**：

基于以下假设：
1. **价格上涨 + 成交量放大** → 主力资金买入
2. **价格下跌 + 成交量放大** → 主力资金卖出
3. **价格平稳 + 成交量放大** → 可能是主力对倒或散户交易

### 2.2 计算方法

#### 步骤1：计算周期内的价格变化和成交量

```python
# 当前周期数据
price_now = df_now['price']           # 当前价格
volume_now = df_now['volume']         # 累计成交量
amount_now = df_now['amount']         # 累计成交额
change_pct_now = df_now['change_pct'] # 涨跌幅

# 上一周期数据（从Redis获取）
price_prev = df_prev['price']
volume_prev = df_prev['volume']
amount_prev = df_prev['amount']

# 周期内变化
delta_price = price_now - price_prev                    # 价格变化
delta_volume = volume_now - volume_prev                 # 成交量变化（股）
delta_amount = amount_now - amount_prev                 # 成交额变化（元）
price_change_pct = (price_now - price_prev) / price_prev * 100  # 价格变化率
```

#### 步骤2：主力资金买入判定

```python
# 买入条件（同时满足）
# 1. 价格上涨（或微跌但成交量极大）
# 2. 成交量放大（超过阈值）
# 3. 成交额超过大单阈值

is_main_buy = (
    (price_change_pct > 0.1) &           # 价格上涨 > 0.1%
    (delta_volume > volume_threshold) &  # 成交量超过阈值
    (delta_amount > amount_threshold)    # 成交额超过大单阈值
)

# 主力资金买入金额 = 符合条件的成交额
main_buy_amount = delta_amount.where(is_main_buy, 0).sum()
```

#### 步骤3：主力资金卖出判定

```python
# 卖出条件（同时满足）
# 1. 价格下跌
# 2. 成交量放大
# 3. 成交额超过大单阈值

is_main_sell = (
    (price_change_pct < -0.1) &          # 价格下跌 > 0.1%
    (delta_volume > volume_threshold) &  # 成交量超过阈值
    (delta_amount > amount_threshold)    # 成交额超过大单阈值
)

# 主力资金卖出金额 = 符合条件的成交额
main_sell_amount = delta_amount.where(is_main_sell, 0).sum()
```

#### 步骤4：主力资金净额

```python
main_net_amount = main_buy_amount - main_sell_amount
```

### 2.3 阈值设计

#### 方案A：固定阈值（简单）

| 阈值类型 | 值 | 说明 |
|----------|-----|------|
| 价格变化阈值 | ±0.1% | 价格变动超过0.1%才计算 |
| 成交量阈值 | 100手（10,000股） | 单笔超过100手算大单 |
| 成交额阈值 | 100,000元 | 单笔超过10万元算大单 |

#### 方案B：动态阈值（推荐）

基于股票流通市值和平均成交量动态计算：

```python
# 动态大单阈值 = 该股票近N日平均成交量的 X%
volume_threshold = avg_volume_20d * 0.05  # 5%的平均成交量

# 动态价格阈值 = 该股票的波动率
price_threshold = std_change_pct_20d * 0.5  # 0.5倍标准差
```

#### 方案C：分级阈值（精细化）

| 股票类型 | 成交量阈值 | 成交额阈值 | 说明 |
|----------|-----------|-----------|------|
| 大盘股（>500亿） | 500手 | 500,000元 | 流通市值大，阈值高 |
| 中盘股（100-500亿） | 200手 | 200,000元 | 中等阈值 |
| 小盘股（<100亿） | 100手 | 100,000元 | 流通市值小，阈值低 |

### 4.2 新增计算函数

```python
def calculate_main_force(df_now: pd.DataFrame, df_prev: pd.DataFrame, 
                         config: dict = None) -> pd.DataFrame:
    """
    计算主力资金流向指标
    
    Args:
        df_now: 当前时刻数据
        df_prev: 上一时刻数据
        config: 配置参数
        
    Returns:
        DataFrame with main force columns
    """
    if config is None:
        config = MAIN_FORCE_CONFIG
    
    # 合并数据
    merged = pd.merge(
        df_now[['stock_code', 'price', 'volume', 'amount', 'change_pct']],
        df_prev[['stock_code', 'price', 'volume', 'amount']],
        on='stock_code',
        suffixes=('_now', '_prev'),
        how='inner'
    )
    
    if merged.empty:
        return pd.DataFrame(columns=['stock_code', 'main_buy_amount', 'main_sell_amount', 
                                     'main_net_amount', 'main_buy_count', 'main_sell_count'])
    
    # 计算变化
    merged['delta_price'] = merged['price_now'] - merged['price_prev']
    merged['delta_volume'] = merged['volume_now'] - merged['volume_prev']
    merged['delta_amount'] = merged['amount_now'] - merged['amount_prev']
    merged['price_change_pct'] = (merged['delta_price'] / merged['price_prev'] * 100).round(2)
    
    # 获取阈值
    price_threshold = config['price_change_threshold']
    volume_threshold = config['volume_threshold_fixed']
    amount_threshold = config['amount_threshold_fixed']
    
    # 主力资金买入判定
    is_main_buy = (
        (merged['price_change_pct'] > price_threshold) &
        (merged['delta_volume'] > volume_threshold) &
        (merged['delta_amount'] > amount_threshold)
    )
    
    # 主力资金卖出判定
    is_main_sell = (
        (merged['price_change_pct'] < -price_threshold) &
        (merged['delta_volume'] > volume_threshold) &
        (merged['delta_amount'] > amount_threshold)
    )
    
    # 计算金额
    merged['main_buy_amount'] = merged['delta_amount'].where(is_main_buy, 0).round(2)
    merged['main_sell_amount'] = merged['delta_amount'].where(is_main_sell, 0).round(2)
    merged['main_net_amount'] = (merged['main_buy_amount'] - merged['main_sell_amount']).round(2)
    
    # 计算次数
    merged['main_buy_count'] = is_main_buy.astype(int)
    merged['main_sell_count'] = is_main_sell.astype(int)
    merged['main_activity'] = merged['main_buy_count'] + merged['main_sell_count']
    
    return merged[['stock_code', 'main_buy_amount', 'main_sell_amount', 
                   'main_net_amount', 'main_buy_count', 'main_sell_count', 'main_activity']]
```
