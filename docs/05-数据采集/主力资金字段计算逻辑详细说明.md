## 2. 各字段计算逻辑

### 2.1 前置数据准备

```python
# 1. 获取当前周期数据（df_now）
price_now = df_now['price']           # 当前价格
volume_now = df_now['volume']         # 累计成交量（当日累计）
amount_now = df_now['amount']         # 累计成交额（当日累计）

# 2. 获取上一周期数据（df_prev，3秒前）
price_prev = df_prev['price']         # 上一时刻价格
volume_prev = df_prev['volume']       # 上一时刻累计成交量
amount_prev = df_prev['amount']       # 上一时刻累计成交额

# 3. 计算周期内变化（3秒周期）
delta_price = price_now - price_prev                    # 价格变化（元）
delta_volume = volume_now - volume_prev                 # 周期成交量（股）
delta_amount = amount_now - amount_prev                 # 周期成交额（元）
price_change_pct = delta_price / price_prev * 100       # 价格变化率（%）
```

---

### 2.2 main_buy_amount（主力资金买入金额）

**计算逻辑**：

```python
# 主力资金买入判定条件（同时满足）
is_main_buy = (
    (price_change_pct > 0.01) &        # 1. 价格上涨 > 0.01%
    (delta_volume >= 20000) &          # 2. 周期成交量 >= 200手
    (delta_amount >= 300000)           # 3. 周期成交额 >= 30万
)

# 计算买入金额
if is_main_buy:
    main_buy_amount = delta_amount      # 该周期的成交额计入主力买入
    main_buy_count = 1                  # 买入次数+1
else:
    main_buy_amount = 0
    main_buy_count = 0
```

**示例**：

| 时间 | price_now | price_prev | delta_price | price_change_pct | delta_volume | delta_amount | is_main_buy | main_buy_amount |
|------|-----------|------------|-------------|------------------|--------------|--------------|-------------|-----------------|
| 09:30:03 | 10.50 | 10.49 | +0.01 | +0.095% | 15,000 | 157,500 | ❌ | 0 |
| 09:30:06 | 10.52 | 10.50 | +0.02 | +0.190% | 25,000 | 263,000 | ❌ | 0 |
| 09:30:09 | 10.55 | 10.52 | +0.03 | +0.285% | 30,000 | 316,500 | ✅ | 316,500 |

### 2.3 main_sell_amount（主力资金卖出金额）

**计算逻辑**：

```python
# 主力资金卖出判定条件（同时满足）
is_main_sell = (
    (price_change_pct < -0.01) &       # 1. 价格下跌 > 0.01%
    (delta_volume >= 20000) &          # 2. 周期成交量 >= 200手
    (delta_amount >= 300000)           # 3. 周期成交额 >= 30万
)

# 计算卖出金额
if is_main_sell:
    main_sell_amount = delta_amount     # 该周期的成交额计入主力卖出
    main_sell_count = 1                 # 卖出次数+1
else:
    main_sell_amount = 0
    main_sell_count = 0
```

### 2.4 main_net_amount（主力资金净额）

**计算逻辑**：

```python
# 净额 = 买入金额 - 卖出金额
main_net_amount = main_buy_amount - main_sell_amount
```

**结果解释**：

| main_net_amount | 含义 |
|-----------------|------|
| > 0 | 主力资金净流入（买入 > 卖出） |
| < 0 | 主力资金净流出（卖出 > 买入） |
| = 0 | 主力资金平衡或无主力交易 |

**示例**：

| 时间 | main_buy_amount | main_sell_amount | main_net_amount | 含义 |
|------|-----------------|------------------|-----------------|------|
| 09:30:09 | 316,500 | 0 | +316,500 | 主力净流入 |
| 10:15:09 | 0 | 364,700 | -364,700 | 主力净流出 |
| 11:00:00 | 250,000 | 180,000 | +70,000 | 主力净流入 |

### 2.5 main_buy_count（主力买入次数）

**计算逻辑**：

```python
# 每个周期独立判断，满足条件则计数+1
main_buy_count = sum(is_main_buy for each period)

# 单日累计（从开盘到当前时间）
main_buy_count_daily = Σ main_buy_count_per_period
```

**示例**（某股票上午交易时段）：

| 09:30:12-09:30:15 | ✅ | 1 | 2 |
| ... | ... | ... | ... |

---

### 2.6 main_sell_count（主力卖出次数）

**计算逻辑**：

```python
# 每个周期独立判断，满足条件则计数+1
main_sell_count = sum(is_main_sell for each period)

# 单日累计（从开盘到当前时间）
main_sell_count_daily = Σ main_sell_count_per_period
```

---

### 2.7 main_activity（主力活跃度）

**计算逻辑**：

```python
# 活跃度 = 买入次数 + 卖出次数
main_activity = main_buy_count + main_sell_count
```

**活跃度分级**：

| main_activity | 活跃度等级 | 说明 |
|---------------|-----------|------|
| 0 | 无 | 无主力交易 |
| 1-5 | 低 | 偶尔有主力参与 |
| 6-15 | 中 | 主力有一定参与度 |
| 16-30 | 高 | 主力积极参与 |
| >30 | 极高 | 主力高度活跃 |

## 4. 阈值敏感性分析

### 4.1 当前阈值（修改后）

| 阈值 | 值 | 敏感性 |
|------|-----|--------|
| 价格变化 | ±0.01% | 极高（价格微小波动即触发） |
| 成交量 | 200手 | 高（中等规模交易） |
| 成交额 | 30万 | 高（中等规模资金） |

### 4.2 预期效果

- **主力买入/卖出次数**：预计增加（阈值降低，更容易触发）
- **主力活跃度**：预计提高
- **信号噪声**：可能增加（需要后续观察调优）

### 4.3 调优建议

如果信号过于频繁，可以：
1. 提高价格阈值到 ±0.05%
2. 提高成交量阈值到 300手
3. 提高成交额阈值到 50万

## 6. 验证方法

### 6.1 数据一致性验证

```python

assert main_net_amount == main_buy_amount - main_sell_amount

# 验证：活跃度 = 买入次数 + 卖出次数
assert main_activity == main_buy_count + main_sell_count

# 验证：非负性
assert main_buy_amount >= 0
assert main_sell_amount >= 0
assert main_buy_count >= 0
assert main_sell_count >= 0
```

### 6.2 业务逻辑验证

```python
# 验证：同时满足买入和卖出的情况（理论上不可能）
assert not (is_main_buy and is_main_sell)

# 验证：价格不变时的处理
if abs(price_change_pct) <= 0.01:
    assert main_buy_amount == 0 and main_sell_amount == 0
```
