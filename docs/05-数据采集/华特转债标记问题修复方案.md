# 华特转债标记问题排查与修复方案

## 问题确认

### 排查结果
| 检查项 | 结果 |
|--------|------|
| 华特科技在股票上攻排行 | ✅ 在（第3位） |
| 华特转债在债券上攻排行 | ✅ 在 |
| 债券实时数据有stock_code | ❌ 无此字段 |
| 债券上攻排行有is_stock_ranked | ❌ 无此字段 |

### 根因
债券数据源（adata）**没有提供正股代码字段**，导致无法建立债券→正股的映射关系。

## 修复方案

### 方案A：通过外部数据源补充正股代码（推荐）

使用AKShare或AData的可转债基本信息表，获取债券→正股的映射。

```python
# 新增：获取债券正股代码映射
def get_bond_stock_mapping() -> dict:
    """
    获取债券代码到正股代码的映射
    
    Returns:
        Dict[债券代码, 正股代码]
    """
    try:
        import akshare as ak
        
        # 方法1：使用AKShare获取可转债列表
        df = ak.bond_cb_jsl()
        if 'bond_id' in df.columns and 'stock_id' in df.columns:
            mapping = dict(zip(
                df['bond_id'].astype(str).str.zfill(6),
                df['stock_id'].astype(str).str.zfill(6)
            ))
            return mapping
    except Exception as e:
        logger.warning(f"AKShare获取债券映射失败: {e}")
    
    try:
        # 方法2：使用AData
        import adata
        df = adata.bond.info.all_bonds()
        if 'bond_code' in df.columns and 'stock_code' in df.columns:
            mapping = dict(zip(
                df['bond_code'].astype(str).str.zfill(6),
                df['stock_code'].astype(str).str.zfill(6)
            ))
            return mapping
    except Exception as e:
        logger.warning(f"AData获取债券映射失败: {e}")
    
    return {}


# 缓存映射关系
_bond_stock_mapping = None

def get_cached_bond_stock_mapping() -> dict:
    """获取缓存的债券正股映射（每天刷新）"""

    if _bond_stock_mapping is None:
        _bond_stock_mapping = get_bond_stock_mapping()
        logger.info(f"[股债联动] 加载债券正股映射: {len(_bond_stock_mapping)}条")
    
    return _bond_stock_mapping
```
