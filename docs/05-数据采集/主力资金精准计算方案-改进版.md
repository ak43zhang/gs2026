## 1. 当前方案的问题

### 1.1 过于简化的逻辑

```python
# 当前问题：只看价格变化
if price_change > 0:
    main_buy = delta_amount  # 所有上涨都算作主力买入
else:
    main_sell = delta_amount  # 所有下跌都算作主力卖出
```

**问题**：
- 散户跟风买入也会导致价格上涨
- 主力出货也可能是缓慢下跌
- 无法区分大单和散户小单

### 1.2 需要解决的问题

1. **如何识别大单交易**（主力特征）
2. **如何识别主动买入vs被动买入**
3. **如何识别吸筹vs出货行为**
4. **如何过滤散户跟风交易**

---

## 2. 改进方案核心思路

### 2.1 多因子识别模型

```
主力资金交易特征：
├── 大单特征
│   ├── 单笔成交额大（>30万）
│   ├── 单笔成交量大（>200手）
│   └── 占该周期成交额比例高（>5%）
│
├── 价格行为特征
│   ├── 主动拉升（价格快速上涨）
│   ├── 托单行为（大单托底）
│   └── 对倒行为（自买自卖）
│
└── 量能特征
    ├── 成交量突增（>均值2倍）
    ├── 量价配合（量增价涨）
    └── 换手率异常（>均值3倍）
```

### 2.2 加权评分模型

```python
# 主力交易评分（0-100分）
main_force_score = (
    amount_score * 0.3 +      # 成交额评分（大单权重）
    volume_score * 0.2 +      # 成交量评分
    price_action_score * 0.3 + # 价格行为评分
    volume_pattern_score * 0.2 # 量能模式评分
)

# 只有评分超过阈值才认定为主力交易
if main_force_score >= 60:
    classify_as_main_force()
```

### 3.2 因子1：成交额评分（amount_score）

```python
def calculate_amount_score(delta_amount, avg_amount_20d):
    """
    成交额评分：判断是否为单笔大单
    """
    # 绝对阈值
    MIN_AMOUNT = 300000  # 30万
    
    # 相对阈值（相对于历史均值）
    amount_ratio = delta_amount / avg_amount_20d if avg_amount_20d > 0 else 0
    
    # 评分计算
    if delta_amount < MIN_AMOUNT:
        # 未达到绝对阈值，可能是散户
        score = 0
    else:
        # 基础分：满足最低门槛
        base_score = 40
        
        # 超额加分
        if delta_amount >= 500000:  # 50万
            base_score += 20
        if delta_amount >= 1000000:  # 100万
            base_score += 20
        if delta_amount >= 2000000:  # 200万
            base_score += 20
            
        # 相对均值加分
        if amount_ratio >= 2:  # 超过2倍均值
            base_score += 10
        if amount_ratio >= 3:  # 超过3倍均值
            base_score += 10
            
        score = min(100, base_score)
    
    return score

# 评分标准
# 0分: <30万（散户级）
# 40分: 30-50万（小单）
# 60分: 50-100万（中单）
# 80分: 100-200万（大单）
# 100分: >200万（超大单）
```

```python
def calculate_volume_score(delta_volume, avg_volume_20d):

    成交量评分：判断成交量是否异常放大

        score = 0
    else:
        base_score = 40
        
        # 手数加分
        if delta_volume >= 50000:   # 500手
            base_score += 15
        if delta_volume >= 100000:  # 1000手
            base_score += 15
        if delta_volume >= 200000:  # 2000手
            base_score += 15
            
        # 倍数加分
        if volume_ratio >= 2:
            base_score += 10
        if volume_ratio >= 3:
            base_score += 10
            
        score = min(100, base_score)
    
    return score
```

### 3.4 因子3：价格行为评分（price_action_score）

```python
def calculate_price_action_score(price_change_pct, delta_price, 
                                  delta_amount, delta_volume,
                                  price_now, avg_volatility_20d):
    """
    价格行为评分：识别主动拉升、托单、对倒等行为
    
    关键逻辑：
    1. 主动拉升：价格快速上涨 + 大单成交
    2. 托单行为：价格稳定 + 大单托底
    3. 对倒行为：价格异常波动 + 自买自卖特征
    """
    score = 0
    
    # === 子因子3.1：价格变化强度 ===
    if price_change_pct > 0.5:  # 快速上涨
        score += 25
    elif price_change_pct > 0.1:  # 正常上涨
        score += 15
    elif price_change_pct < -0.5:  # 快速下跌
        score += 25  # 可能是主力出货
    elif price_change_pct < -0.1:  # 正常下跌
        score += 15
    
    # === 子因子3.2：量价配合度 ===
    # 量价齐升 = 主力主动买入
    # 价跌量增 = 主力主动卖出
    if price_change_pct > 0.05 and delta_volume > 0:
        score += 20  # 量价齐升
    elif price_change_pct < -0.05 and delta_volume > 0:
        score += 20  # 价跌量增
    
    # === 子因子3.3：价格波动异常度 ===
    # 如果价格波动远超正常波动率，可能是主力操纵
    if avg_volatility_20d > 0:
        volatility_ratio = abs(price_change_pct) / avg_volatility_20d
        if volatility_ratio >= 3:  # 超过3倍正常波动
            score += 15
        elif volatility_ratio >= 2:  # 超过2倍正常波动
            score += 10
    
    # === 子因子3.4：价格位置分析 ===
    # 如果价格处于关键位置（整数关口、前高、前低），主力行为概率高
    # 简化处理：价格末位接近整数
    price_decimal = price_now % 1
    if price_decimal < 0.02 or price_decimal > 0.98:
        score += 10  # 接近整数关口
    
    return min(100, score)
```

### 3.5 因子4：量能模式评分（volume_pattern_score）

```python
def calculate_volume_pattern_score(delta_volume, delta_amount,
                                    avg_volume_20d, avg_amount_20d,
                                    price_change_pct):
    """
    量能模式评分：识别量能异常和主力行为模式
    """
    score = 0
    
    # === 子因子4.1：量能突变 ===
    if avg_volume_20d > 0:
        volume_surge = delta_volume / avg_volume_20d
        if volume_surge >= 5:  # 5倍放量
            score += 30
        elif volume_surge >= 3:  # 3倍放量
            score += 20
        elif volume_surge >= 2:  # 2倍放量
            score += 10
    
    # === 子因子4.2：换手率异常 ===
    # 换手率 = 成交量 / 流通股本
    # 这里用成交额占比代替
    if avg_amount_20d > 0:
        turnover_ratio = delta_amount / avg_amount_20d
        if turnover_ratio >= 0.1:  # 占日均成交10%
            score += 20
        elif turnover_ratio >= 0.05:  # 占日均成交5%
            score += 10
    
    # === 子因子4.3：量价背离检测 ===
    # 价涨量缩 = 上涨乏力（散户追高）
    # 价跌量缩 = 下跌乏力（散户割肉）
    # 这两种情况主力参与度低
    if price_change_pct > 0.1 and delta_volume < avg_volume_20d * 0.5:
        # 价涨量缩，散户行为
        score -= 15
    elif price_change_pct < -0.1 and delta_volume < avg_volume_20d * 0.5:
        # 价跌量缩，散户行为
        score -= 15
    else:
        # 量价配合正常
        score += 10
    
    return max(0, min(100, score))
```

### 3.6 综合评分与分类

```python
def calculate_main_force_indicator(stock_data, historical_data):
    """
    计算主力资金指标
    
    Returns:
        {
            'main_force_score': 85,      # 主力评分（0-100）
            'is_main_force': True,       # 是否认定为主力交易
            'direction': 'buy',          # 方向：buy/sell
            'confidence': 'high',        # 置信度：high/medium/low
            'main_buy_amount': 500000,   # 主力买入金额
            'main_sell_amount': 0,       # 主力卖出金额
            'main_net_amount': 500000,   # 主力净额
        }
    """
    # 计算各因子得分
    amount_score = calculate_amount_score(delta_amount, avg_amount_20d)
    volume_score = calculate_volume_score(delta_volume, avg_volume_20d)
    price_action_score = calculate_price_action_score(...)
    volume_pattern_score = calculate_volume_pattern_score(...)
    
    # 加权综合评分
    main_force_score = (
        amount_score * 0.35 +        # 成交额权重最高
        volume_score * 0.20 +
        price_action_score * 0.30 +
        volume_pattern_score * 0.15
    )
    
    # 判定是否为主力交易
    is_main_force = main_force_score >= 60
    
    # 判定方向
    if is_main_force:
        if price_change_pct > 0:
            direction = 'buy'
            main_buy_amount = delta_amount * (main_force_score / 100)
            main_sell_amount = 0
        else:
            direction = 'sell'
            main_buy_amount = 0
            main_sell_amount = delta_amount * (main_force_score / 100)
    else:
        direction = 'neutral'
        main_buy_amount = 0
        main_sell_amount = 0
    
    # 计算净额
    main_net_amount = main_buy_amount - main_sell_amount
    
    # 置信度
    if main_force_score >= 80:
        confidence = 'high'
    elif main_force_score >= 60:
        confidence = 'medium'
    else:
        confidence = 'low'
    
    return {
        'main_force_score': round(main_force_score, 2),
        'is_main_force': is_main_force,
        'direction': direction,
        'confidence': confidence,
        'main_buy_amount': round(main_buy_amount, 2),
        'main_sell_amount': round(main_sell_amount, 2),
        'main_net_amount': round(main_net_amount, 2),
        # 明细用于调试
        'detail': {
            'amount_score': amount_score,
            'volume_score': volume_score,
            'price_action_score': price_action_score,
            'volume_pattern_score': volume_pattern_score
        }
    }
```
