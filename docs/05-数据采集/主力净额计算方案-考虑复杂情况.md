# 主力净额字段计算方案（考虑复杂情况）

> 更新时间：2026-04-28 13:00  
> 目标：解决价格上涨但主力净额为负（拉高出货）、价格下跌但主力净额为正（打压吸筹）的情况

## 1. 问题分析

### 1.1 当前方案的问题

```python
# 当前方案的问题：只看价格涨跌判断方向
def calculate_direction_coefficient(price_change_pct):
    if price_change_pct > 0:
        return 1.0  # 价格上涨 = 主力买入

# 问题场景：

# - 实际：主力在高位大量卖出
# - 当前方案误判：main_net = +200万（错误）
# - 实际情况：main_net = -300万（主力净流出）

# 场景2：主力打压吸筹
# - 价格：-2%（大幅下跌）
# - 实际：主力在低位大量买入
# - 当前方案误判：main_net = -200万（错误）
# - 实际情况：main_net = +300万（主力净流入）
```

### 1.2 主力行为的复杂性

```
主力行为模式：
├── 价格上涨时的主力行为
│   ├── 主动拉升（买入）→ 主力净额为正
│   ├── 拉高出货（卖出）→ 主力净额为负
│   └── 散户跟风（观望）→ 主力净额接近0
│
├── 价格下跌时的主力行为
│   ├── 恐慌抛售（卖出）→ 主力净额为负
│   ├── 打压吸筹（买入）→ 主力净额为正
│   └── 散户割肉（观望）→ 主力净额接近0
│
└── 价格平稳时的主力行为
    ├── 横盘吸筹（买入）→ 主力净额为正（小）
    ├── 横盘出货（卖出）→ 主力净额为负（小）
    └── 无主力参与 → 主力净额为0
```

### 1.3 核心问题

```
问题：仅用价格变化无法判断主力真实意图

需要额外的判断依据：
1. 价格位置（高位/低位）
2. 量能特征（放量/缩量）
3. 价格变化速度（急涨/缓涨）
4. 历史行为模式
5. 盘口压力（买卖力量对比）
```

## 3. 详细计算逻辑

### 3.1 价格位置判断

```python
def calculate_price_position(price_now, price_history):
    """
    判断当前价格位置（高位/中位/低位）
    
    基于当日价格区间：
    - 高位：当前价格 > 当日最高价 × 0.95
    - 低位：当前价格 < 当日最低价 × 1.05
    - 中位：其他情况
    """
    day_high = price_history['day_high']
    day_low = price_history['day_low']
    day_open = price_history['day_open']
    
    price_range = day_high - day_low
    if price_range == 0:
        return 'mid'
    
    # 计算当前价格在当日区间的位置（0-1）
    position = (price_now - day_low) / price_range
    
    if position >= 0.95:  # 接近最高价
        return 'high'
    elif position <= 0.05:  # 接近最低价
        return 'low'
    elif position >= 0.7:  # 偏高
        return 'high_mid'
    elif position <= 0.3:  # 偏低
        return 'low_mid'
    else:
        return 'mid'

# 价格位置系数
PRICE_POSITION_FACTOR = {
    'high': {'bias': 'sell', 'weight': 1.0},      # 高位倾向卖出
    'high_mid': {'bias': 'sell', 'weight': 0.5},  # 偏高倾向卖出
    'mid': {'bias': 'neutral', 'weight': 0},      # 中位中性
    'low_mid': {'bias': 'buy', 'weight': 0.5},    # 偏低倾向买入
    'low': {'bias': 'buy', 'weight': 1.0},        # 低位倾向买入
}
```

### 3.2 量能特征判断

```python
def calculate_volume_characteristic(delta_volume, delta_amount, 
                                    avg_volume, avg_amount):
    """
    判断量能特征
    
    返回：
    - volume_type: 'high'/'normal'/'low'
    - volume_ratio: 相对于均值的倍数
    """
    if avg_volume <= 0 or avg_amount <= 0:
        return 'normal', 1.0
    
    volume_ratio = delta_volume / avg_volume
    amount_ratio = delta_amount / avg_amount
    
    if volume_ratio >= 3 or amount_ratio >= 3:
        return 'extreme_high', max(volume_ratio, amount_ratio)
    elif volume_ratio >= 2 or amount_ratio >= 2:
        return 'high', max(volume_ratio, amount_ratio)
    elif volume_ratio >= 1.5 or amount_ratio >= 1.5:
        return 'medium_high', max(volume_ratio, amount_ratio)
    elif volume_ratio <= 0.5 or amount_ratio <= 0.5:
        return 'low', min(volume_ratio, amount_ratio)
    else:
        return 'normal', 1.0
```

### 3.3 价格速度判断

```python
def calculate_price_speed(price_change_pct, time_duration=3):
    """
    计算价格变化速度
    
    返回：
    - speed: 'fast'/'normal'/'slow'
    - speed_score: 速度评分（0-1）
    """
    # 计算每秒变化率
    change_per_second = abs(price_change_pct) / time_duration
    
    # A股正常波动：±0.1%/秒 以内
    # 快速变化：>0.3%/秒
    if change_per_second >= 0.3:  # 每秒变化>0.3%
        return 'fast', min(1.0, change_per_second / 0.5)
    elif change_per_second >= 0.1:  # 每秒变化0.1%-0.3%
        return 'normal', 0.5 + (change_per_second - 0.1) / 0.2 * 0.5
    else:  # 每秒变化<0.1%
        return 'slow', change_per_second / 0.1 * 0.5
```

### 3.4 综合行为判断

```python
def classify_main_force_behavior(price_change_pct, price_position, 
                                 volume_char, price_speed):
    """
    综合判断主力行为
    
    返回：
    - behavior: 行为类型
    - direction: 方向（1=买入，-1=卖出，0=中性）
    - confidence: 置信度（0-1）
    """
    
    # 情况1：高位 + 放量 + 上涨 = 拉高出货
    if price_position in ['high', 'high_mid'] and \
       volume_char in ['high', 'extreme_high'] and \
       price_change_pct > 0.3:
        return {
            'behavior': '拉高出货',
            'direction': -1,  # 主力在卖出
            'confidence': 0.85 if price_position == 'high' else 0.7
        }
    
    # 情况2：低位 + 放量 + 下跌 = 打压吸筹
    elif price_position in ['low', 'low_mid'] and \
         volume_char in ['high', 'extreme_high'] and \
         price_change_pct < -0.3:
        return {
            'behavior': '打压吸筹',
            'direction': 1,  # 主力在买入
            'confidence': 0.85 if price_position == 'low' else 0.7
        }
    
    # 情况3：低位 + 放量 + 上涨 = 主动拉升
    elif price_position in ['low', 'low_mid'] and \
         volume_char in ['high', 'extreme_high'] and \
         price_change_pct > 0.3:
        return {
            'behavior': '主动拉升',
            'direction': 1,  # 主力在买入
            'confidence': 0.8
        }
    
    # 情况4：高位 + 放量 + 下跌 = 恐慌抛售/主力出货
    elif price_position in ['high', 'high_mid'] and \
         volume_char in ['high', 'extreme_high'] and \
         price_change_pct < -0.3:
        return {
            'behavior': '恐慌抛售',
            'direction': -1,  # 主力在卖出
            'confidence': 0.75
        }
    
    # 情况5：中位 + 放量 + 上涨 = 正常拉升
    elif price_position == 'mid' and \
         volume_char in ['high', 'extreme_high'] and \
         price_change_pct > 0.5:
        return {
            'behavior': '正常拉升',
            'direction': 1,
            'confidence': 0.7
        }
    
    # 情况6：中位 + 放量 + 下跌 = 正常回调
    elif price_position == 'mid' and \
         volume_char in ['high', 'extreme_high'] and \
         price_change_pct < -0.5:
        return {
            'behavior': '正常回调',
            'direction': -1,
            'confidence': 0.6
        }
    
    # 情况7：任何位置 + 缩量 + 上涨 = 散户跟风
    elif volume_char == 'low' and price_change_pct > 0:
        return {
            'behavior': '散户跟风',
            'direction': 0,  # 主力未参与
            'confidence': 0.6
        }
    
    # 情况8：任何位置 + 缩量 + 下跌 = 散户割肉
    elif volume_char == 'low' and price_change_pct < 0:
        return {
            'behavior': '散户割肉',
            'direction': 0,  # 主力未参与
            'confidence': 0.6
        }
    
    # 情况9：横盘 + 放量 = 主力调仓（方向不确定）
    elif abs(price_change_pct) <= 0.1 and volume_char in ['high', 'extreme_high']:
        # 根据价格位置判断
        if price_position in ['high', 'high_mid']:
            return {
                'behavior': '高位调仓出货',
                'direction': -0.5,  # 偏卖出
                'confidence': 0.5
            }
        elif price_position in ['low', 'low_mid']:
            return {
                'behavior': '低位调仓吸筹',
                'direction': 0.5,  # 偏买入
                'confidence': 0.5
            }
        else:
            return {
                'behavior': '横盘调仓',
                'direction': 0,  # 中性
                'confidence': 0.3
            }
    
    # 默认情况
    else:
        return {
            'behavior': '正常交易',
            'direction': 1 if price_change_pct > 0 else -1,
            'confidence': 0.4
        }
```

### 3.5 主力净额计算

```python
def calculate_main_net_amount_v4(stock_code, current_data, previous_data, 
                               historical_data, day_stats):
    """
    计算主力净额（改进版，考虑复杂情况）
    
    公式：主力净额 = 周期成交额 × 主力参与系数 × 方向系数 × 置信度
    """
    
    # 计算周期变化
    delta_amount = current_data['amount'] - previous_data['amount']
    delta_volume = current_data['volume'] - previous_data['volume']
    price_change_pct = current_data['change_pct'] - previous_data.get('change_pct', 0)
    price_now = current_data['price']
    
    # 门槛检查
    MIN_AMOUNT = 300000
    MIN_VOLUME = 20000
    if delta_amount < MIN_AMOUNT or delta_volume < MIN_VOLUME:
        return 0.0
    
    # 获取历史统计
    avg_amount = historical_data.get('avg_amount_20d', delta_amount)
    avg_volume = historical_data.get('avg_volume_20d', delta_volume)
    
    # 获取当日统计
    day_high = day_stats.get('day_high', price_now)
    day_low = day_stats.get('day_low', price_now)
    day_open = day_stats.get('day_open', price_now)
    
    price_history = {
        'day_high': day_high,
        'day_low': day_low,
        'day_open': day_open
    }
    
    # 1. 计算价格位置
    price_position = calculate_price_position(price_now, price_history)
    
    # 2. 计算量能特征
    volume_char, volume_ratio = calculate_volume_characteristic(
        delta_volume, delta_amount, avg_volume, avg_amount
    )
    
    # 3. 计算价格速度
    price_speed, speed_score = calculate_price_speed(price_change_pct)
    
    # 4. 判断主力行为
    behavior = classify_main_force_behavior(
        price_change_pct, price_position, volume_char, price_speed
    )
    
    # 5. 计算主力参与系数（基于成交额）
    if delta_amount >= 2000000:
        participation = 1.0
    elif delta_amount >= 1000000:
        participation = 0.8 + (delta_amount - 1000000) / 1000000 * 0.2
    elif delta_amount >= 500000:
        participation = 0.5 + (delta_amount - 500000) / 500000 * 0.3
    else:
        participation = 0.3 + (delta_amount - 300000) / 200000 * 0.2
    
    # 量能放大加成
    if volume_ratio > 1:
        participation = min(1.0, participation + min(0.2, (volume_ratio - 1) * 0.1))
    
    # 6. 计算主力净额
    direction = behavior['direction']  # -1, -0.5, 0, 0.5, 1
    confidence = behavior['confidence']
    
    if direction != 0:
        # 正常情况
        main_net = delta_amount * participation * direction * confidence
    else:
        # 方向中性，但可能是主力在对倒或观望
        # 保守处理：只计算参与系数的一半
        main_net = delta_amount * participation * 0.3
    
    return {
        'main_net_amount': round(main_net, 2),
        'behavior': behavior['behavior'],
        'direction': direction,
        'confidence': confidence,
        'price_position': price_position,
        'volume_char': volume_char,
        'participation': round(participation, 2)
    }
```

## 6. 计算示例

### 6.1 拉高出货场景

```
数据：
- 当前价格：12.50元
- 当日最高：12.60元
- 当日最低：11.00元
- 价格位置：high（接近最高价）
- 价格变化：+2.0%
- 成交额：500万
- 成交量：是均值的4倍

判断：
- 价格位置：high（高位）
- 量能：extreme_high（极端放量）
- 价格变化：上涨
- 行为识别：拉高出货
- 方向：-1（卖出）
- 置信度：0.85

计算：
- 参与系数：1.0（500万）
- 主力净额 = 500万 × 1.0 × (-1) × 0.85 = -425万

结果：主力净流出425万（拉高出货）
```
