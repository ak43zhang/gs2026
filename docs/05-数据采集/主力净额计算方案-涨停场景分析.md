## 1. 涨停情况分析

### 1.1 涨停的两种类型

```
涨停类型：
├── 类型A：主力主动封板（买入）
│   ├── 特征：
│   │   ├── 早盘快速涨停
│   │   ├── 封板坚决，封单大
│   │   ├── 成交量适中（不是天量）
│   │   └── 后续不开板
│   └── 主力意图：看好后市，主动买入
│
└── 类型B：主力出货型涨停（卖出）
    ├── 特征：
    │   ├── 尾盘偷袭涨停
    │   ├── 反复开板，封单小
    │   ├── 成交量巨大（天量）
    │   └── 次日低开
    └── 主力意图：拉高出货，吸引跟风

关键问题：
- 涨停时价格不再变化（change_pct固定）
- 如何判断主力是在买入还是卖出？
- 主力净额应该是正还是负？
```

### 1.2 涨停时的数据特征


## 2. 涨停情况下的判断逻辑

### 2.1 基于可用数据的判断

```python
def classify_zt_behavior(stock_code, zt_data, context):
    """
    判断涨停时的主力行为
    
    基于可用数据：
    - 封板时间
    - 封板前走势
    - 封板后成交量
    """
    
    # ===== 因子1：封板时间 =====
    zt_time = zt_data['time']  # 首次涨停时间
    
    if dt_time(9, 30) <= zt_time <= dt_time(10, 0):
        time_factor = '早盘涨停'
        time_score = 0.8  # 早盘涨停，主力买入概率高
    elif dt_time(10, 0) <= zt_time <= dt_time(13, 0):
        time_factor = '盘中涨停'
        time_score = 0.6
    else:
        time_factor = '尾盘涨停'
        time_score = 0.3  # 尾盘涨停，出货概率增加
    
    # ===== 因子2：封板前走势 =====
    pre_zt_trend = context['pre_zt_trend']  # 封板前30分钟走势
    
    if pre_zt_trend == 'steady_rise':
        # 稳步上涨后涨停 → 主力吸筹后拉升
        trend_score = 0.8
        trend_factor = '稳步拉升'
    elif pre_zt_trend == 'rapid_rise':
        # 快速拉升后涨停 → 可能是出货
        trend_score = 0.4
        trend_factor = '快速拉升'
    elif pre_zt_trend == 'volatile':
        # 震荡后涨停 → 不确定
        trend_score = 0.5
        trend_factor = '震荡涨停'
    else:
        trend_score = 0.5
        trend_factor = '其他'
    
    # ===== 因子3：封板后成交量 =====
    post_zt_volume = zt_data['post_zt_volume']  # 封板后成交量
    
    if post_zt_volume == 0:
        # 一字板，无成交 → 强烈看好
        volume_score = 1.0
        volume_factor = '一字板'
    elif post_zt_volume < context['avg_volume'] * 0.5:
        # 封板后缩量 → 筹码锁定良好
        volume_score = 0.8
        volume_factor = '缩量封板'
    elif post_zt_volume <= context['avg_volume'] * 2:
        # 封板后正常放量 → 正常换手
        volume_score = 0.6
        volume_factor = '正常换手'
    else:
        # 封板后巨量 → 可能是出货
        volume_score = 0.3
        volume_factor = '巨量封板'
    
    # ===== 综合判断 =====
    total_score = (time_score + trend_score + volume_score) / 3
    
    if total_score >= 0.7:
        behavior = '主力封板'
        direction = 1  # 买入
        confidence = total_score
    elif total_score <= 0.4:
        behavior = '拉高出货'
        direction = -1  # 卖出
        confidence = 1 - total_score
    else:
        behavior = '不确定'
        direction = 0.5  # 偏买入
        confidence = 0.5
    
    return {
        'behavior': behavior,
        'direction': direction,
        'confidence': confidence,
        'factors': {
            'time': time_factor,
            'trend': trend_factor,
            'volume': volume_factor
        }
    }
```

### 2.2 涨停时的主力净额计算

```python
def calculate_main_net_at_zt(stock_code, zt_data, context):
    """
    计算涨停时的主力净额
    
    关键：涨停期间的成交额归属
    """
    
    # 封板期间的成交额
    zt_amount = zt_data['post_zt_amount']
    
    if zt_amount == 0:
        # 一字板，无成交
        return 0.0
    
    # 判断主力行为
    classification = classify_zt_behavior(stock_code, zt_data, context)
    
    # 参与系数（涨停期间成交都是大单）
    participation = 1.0
    
    # 方向
    direction = classification['direction']
    confidence = classification['confidence']
    
    # 计算主力净额
    # 涨停期间的成交：
    # - 如果是主力封板：主力在吃掉卖单（买入）
    # - 如果是出货：主力在卖出给散户
    main_net = zt_amount * participation * direction * confidence
    
    return {
        'main_net_amount': round(main_net, 2),
        'behavior': classification['behavior'],
        'direction': direction,
        'confidence': confidence
    }
```

## 3. 涨停情况下的主力净额方向

### 3.1 不同涨停类型的主力净额

| 涨停类型 | 主力行为 | 主力净额方向 | 说明 |
|----------|----------|--------------|------|
| 早盘一字板 | 主力强烈看好 | 正（买入） | 主力抢筹，无卖盘 |
| 早盘放量涨停 | 主力主动封板 | 正（买入） | 主力吃掉所有卖单 |
| 盘中稳步涨停 | 主力拉升 | 正（买入） | 主力主导上涨 |
| 尾盘偷袭涨停 | 主力拉高出货 | 负（卖出） | 主力吸引跟风后出货 |
| 反复开板涨停 | 主力出货 | 负（卖出） | 主力在板上卖出 |
| 巨量涨停 | 主力对倒/出货 | 不确定 | 可能是出货 |

### 3.2 关键判断依据

```
判断涨停主力意图的关键：

1. 封板时间（最重要）
   - 早盘（9:30-10:00）：主力买入概率 80%
   - 盘中（10:00-14:00）：主力买入概率 60%
   - 尾盘（14:30-15:00）：主力出货概率 70%

2. 封板前走势
   - 稳步上涨：主力买入
   - 快速拉升：可能是出货

3. 封板后成交量
   - 缩量：筹码锁定，主力持有
   - 巨量：主力在出货

4. 开板次数（需要逐笔数据）
   - 不开板：主力坚决
   - 反复开板：主力出货
```

## 5. 最终方案（涨停处理）

### 5.1 涨停时的简化判断

```python
def simple_zt_classify(zt_time, pre_zt_volume, post_zt_volume, avg_volume):
    """
    涨停时的简化判断
    """
    
    # 早盘涨停（9:30-10:00）
    if dt_time(9, 30) <= zt_time <= dt_time(10, 0):
        if post_zt_volume <= avg_volume * 0.5:
            # 早盘涨停 + 缩量 → 主力强烈买入
            return {'behavior': '早盘缩量涨停', 'direction': 1, 'confidence': 0.9}
        else:
            # 早盘涨停 + 放量 → 主力买入
            return {'behavior': '早盘放量涨停', 'direction': 1, 'confidence': 0.8}
    
    # 盘中涨停（10:00-14:30）
    elif dt_time(10, 0) <= zt_time <= dt_time(14, 30):
        if pre_zt_volume > avg_volume * 3:
            # 盘中快速拉升涨停 → 可能是出货
            return {'behavior': '盘中急拉涨停', 'direction': -0.5, 'confidence': 0.5}
        else:
            # 正常涨停
            return {'behavior': '盘中涨停', 'direction': 1, 'confidence': 0.6}
    
    # 尾盘涨停（14:30-15:00）
    else:
        if post_zt_volume > avg_volume * 3:
            # 尾盘涨停 + 巨量 → 出货
            return {'behavior': '尾盘巨量涨停', 'direction': -1, 'confidence': 0.75}
        else:
            # 尾盘涨停 → 偏向出货
            return {'behavior': '尾盘涨停', 'direction': -0.7, 'confidence': 0.6}
```

### 5.2 涨停时的主力净额

```python
def calculate_zt_main_net(zt_data, classification):
    """
    涨停时的主力净额计算
    """
    
    # 涨停期间的成交额
    zt_amount = zt_data['post_zt_amount']
    
    if zt_amount == 0:
        return 0.0
    
    # 涨停期间的成交都视为大单
    participation = 1.0
    
    # 方向和置信度
    direction = classification['direction']
    confidence = classification['confidence']
    
    # 计算
    main_net = zt_amount * participation * direction * confidence
    
    return round(main_net, 2)
```

## 6. 总结

### 6.1 涨停时的主力净额方向

| 涨停类型 | 主力净额 | 说明 |
|----------|----------|------|
| 早盘缩量涨停 | 正（大） | 主力强烈买入 |
| 早盘放量涨停 | 正（中） | 主力买入 |
| 盘中正常涨停 | 正（小） | 主力买入 |
| 盘中急拉涨停 | 负（小） | 可能是出货 |
| 尾盘涨停 | 负（中） | 偏向出货 |
| 尾盘巨量涨停 | 负（大） | 主力出货 |

### 6.2 关键结论

```
1. 涨停不总是主力买入
   - 早盘涨停：主力买入概率高
   - 尾盘涨停：主力出货概率高

2. 需要结合时间判断
   - 时间是最重要的判断因子

3. 成交量辅助判断
   - 缩量涨停：主力持有
   - 巨量涨停：主力出货

4. 准确率限制
   - 由于数据限制，准确率约70%
   - 无法识别反复开板等情况
```
