## 技术总结

### Spark技术
| 技术点 | 应用 |
|--------|------|
| Window函数 | `lag()`获取前N天，`lead()`获取下一个交易日 |
| 日期计算 | `date_add()`，交易日历关联 |
| 数据关联 | `join()`关联交易日历表 |
| 条件筛选 | `where()`多条件组合 |

### 缓存技术
| 技术点 | 应用 |
|--------|------|
| Redis Set | `sadd`、`smembers`实现O(1)查询 |
| 缓存预热 | 同步/异步模式，优先级管理 |
| 缓存管理器 | 统一注册、自动预热 |
