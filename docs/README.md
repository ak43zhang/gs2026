## 文档维护规范

1. **新增文档**：按内容分类放入对应目录，使用中文命名
2. **文档命名**：使用简洁明了的中文名称，如"债券采集优化.md"
3. **交叉引用**：使用相对路径链接，如 `[性能优化计划](02-性能优化/性能优化计划.md)`
4. **更新记录**：重要文档在顶部添加更新历史
