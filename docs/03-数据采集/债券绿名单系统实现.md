# 债券绿名单系统实现文档

## 项目背景

开发债券绿名单系统，用于筛选符合条件的可转债作为买入信号参考。系统包含Spark批处理程序、缓存系统和前端展示。

## 需求分析

### 筛选条件

#### 条件1（model=1）
- **规则**: 当前交易日最高涨幅(zgzf) > 4
- **买入日**: 下一个交易日
- **用途**: 捕捉强势突破信号

#### 条件2（model=2）
- **规则**: 当前交易日和前一个交易日的实体涨幅(stzf)之和 > 4，且两天均 > 0
- **买入日**: 下一个交易日
- **用途**: 捕捉连续上涨动能

### 时间逻辑

```
数据日期: 2026-04-09 (周三)
下一个交易日: 2026-04-10 (周四，根据交易日历)

条件判断基于: 2026-04-09 的数据
买入日期: 2026-04-10
```

#### 关键代码

```scala
// 读取交易日历，计算下一个交易日
val calendarDF = spark.read.jdbc(url, "data_jyrl", properties)
  .where("trade_status = 1")
  .withColumn("trade_date", col("trade_date").cast("date"))

val nextTradeDateDF = calendarDF
  .withColumn("next_trade_date", lead("trade_date", 1).over(Window.orderBy("trade_date")))
  .select(col("trade_date").alias("curr_date"), col("next_trade_date").alias("next_date"))

// 关联债券数据
val bondWithNextDate = bondDF
  .join(nextTradeDateDF, bondDF("date") === nextTradeDateDF("curr_date"), "left")
  .withColumn("prev1_stzf", lag("stzf", 1).over(Window.partitionBy("bond_code").orderBy("date")))

// 条件1: zgzf > 4
val condition1DF = bondWithNextDate
  .where("zgzf > 4")
  .select(col("bond_code").alias("code"), col("next_date").alias("buy_date"), lit("1").alias("model"))

// 条件2: stzf + prev1_stzf > 4，且均 > 0
val condition2DF = bondWithNextDate
  .where("stzf > 0 AND prev1_stzf > 0 AND (stzf + prev1_stzf) > 4")
  .select(col("bond_code").alias("code"), col("next_date").alias("buy_date"), lit("2").alias("model"))
```

```python
def update_green_bond_list_cache(date_str: str = None) -> dict:
    """更新绿名单缓存"""
    # 查询MySQL
    sql = f"SELECT DISTINCT code FROM green_bond_list WHERE buy_date='{date_sql}'"
    df = pd.read_sql(sql, con=mysql_tool.engine)
    
    # 存储到Redis Set
    client = redis_util._get_redis_client()
    set_key = f"{REDIS_KEY}:set"
    codes = df['code'].astype(str).str.zfill(6).tolist()
    client.sadd(set_key, *codes)
    client.expire(set_key, 7 * 24 * 3600)

def get_green_bond_list() -> Set[str]:
    """获取绿名单代码集合"""
    client = redis_util._get_redis_client()
    codes = client.smembers(f"{REDIS_KEY}:set")
    return {code.decode('utf-8') for code in codes}
```

## 数据库设计

### green_bond_list表

```sql
CREATE TABLE green_bond_list (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    code VARCHAR(10) NOT NULL COMMENT '债券代码',
    buy_date DATE NOT NULL COMMENT '期望买入日期',
    model VARCHAR(10) NOT NULL COMMENT '条件编号(1或2)',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_buy_date (buy_date),
    INDEX idx_code (code),
    UNIQUE KEY uk_code_date (code, buy_date)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='债券绿名单';
```

### data_jyrl表（交易日历）

```sql
CREATE TABLE data_jyrl (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    trade_date DATE NOT NULL COMMENT '日期',
    trade_status INT NOT NULL DEFAULT 1 COMMENT '1=交易日,0=非交易日',
    day_week INT COMMENT '星期几(0-6)',
    UNIQUE KEY uk_date (trade_date)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='交易日历';
```
